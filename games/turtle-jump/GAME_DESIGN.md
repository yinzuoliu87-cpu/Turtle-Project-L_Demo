# 龟龟出海 — Bubble Jump to Freedom

## 游戏需求文档

---

## 1. 游戏概述

**游戏名称：** 龟龟出海（Bubble Jump to Freedom）

**游戏类型：** 纵向跳跃类休闲小游戏

**平台：** Web（移动端 + 桌面端自适应）

**技术栈：** 原生 HTML5 Canvas + JavaScript，无外部框架依赖

**核心玩法：** 玩家操控一只小海龟，通过瞄准角度跳跃气泡，从海底逐层上升，穿越 50 层气泡到达海面即为通关。

---

## 2. 核心游戏循环

### 2.1 游戏状态

| 状态 | 说明 |
|------|------|
| `start` | 开始界面，展示角色选择、排行榜、历史最佳 |
| `aiming` | 瞄准阶段：角度指示器自动左右摆动，等待玩家操作 |
| `flying` | 飞行阶段：海龟按当前角度弹射，受重力影响做抛物线运动 |
| `dead` | 死亡/结算：展示成绩、奖励、排行 |
| `win` | 通关：到达海面，进入结算 |

### 2.2 操作方式

- **桌面端：** 空格键 / 上箭头 / 点击 Canvas
- **移动端：** 触摸屏幕
- 操作时机：在 `aiming` 阶段按下即跳跃

---

## 3. 物理与参数

| 参数 | 值 | 说明 |
|------|----|------|
| 气泡总数 | 50 | 固定 50 层气泡 |
| 气泡半径 | 40px | 基础半径，部分阶段有变化 |
| 气泡间距 | 175px | 相邻气泡的世界坐标距离 |
| 海面高度 | 8750px | 50 × 175，到达即通关 |
| 重力 | 0.17 | 每帧下降加速度 |
| 跳跃力 | 9.8 | 弹射初速度 |
| 角度范围 | ±70°（±1.22 rad） | 摆动极限 |
| 角度摆速 | 0.028 基础值 | 随进度增加 (×1.0 ~ ×1.6) |
| 海龟碰撞半径 | 18px | 用于落地和障碍检测 |
| 墙壁宽度 | 28px | 48%+ 进度后出现 |

## 4. 关卡阶段设计

游戏以进度百分比划分为多个阶段，每阶段引入新的障碍和机制：

### 4.1 阶段划分

| 进度 | 区域名称 | 气泡特性 | 新增机制 |
|------|----------|----------|----------|
| 0–5% | 海底 | 居中固定 | 新手引导，显示"点击跳跃" |
| 5–20% | 浅海 | 随机位置，10%+ 开始缓慢移动 | 水流变化提示 |
| 20–30% | 中层 | 移动加快，范围加大 | 碎裂气泡（30%概率） |
| 30–38% | 平静区（喘息） | 气泡加大 ×1.15，移动减缓 | 休息节奏 |
| 38–55% | 暗流区 | 移动快，岩石障碍（45%概率） | 水母、水流推力 |
| 55–62% | 平静区（喘息） | 同第一个平静区 | 休息节奏 |
| 62–80% | 深海 | 高速移动，岩石（50%概率） | 所有机制叠加 |
| 80–90% | 海面 | 高速移动，岩石（35%概率） | 墙壁反弹 |
| 90–100% | 冲刺 | 气泡缩小至 ×0.5，移动极快 | 木板封路障碍 |

### 4.2 喘息区设计

在 30–38% 和 55–62% 两个区间设置"平静区"：
- 气泡半径增大 15%
- 移动速度降低
- 不生成岩石
- 不生成碎裂气泡

---

## 5. 障碍系统

### 5.1 碎裂气泡

- **分布区间：** 20–60%（排除喘息区）
- **出现概率：** 30%
- **机制：** 海龟落上后启动 7 秒倒计时，时间到气泡破碎，海龟掉回上一个气泡
- **视觉反馈：**
  - 未激活时有细微裂纹提示
  - 激活后颜色从蓝渐变至红
  - 顶部显示倒计时进度条（蓝→黄→红）
  - 裂纹线条随时间增多

### 5.2 岩石障碍

- **分布区间：** 38–90%
- **出现概率：** 35–50%（随阶段变化）
- **机制：** 位于气泡下方，阻挡海龟从下方进入，碰撞后弹回（速度 ×0.65）

### 5.3 木板封路（90%+）

- **分布区间：** 90–100%
- **出现概率：** 60%
- **组成：**
  - 底部封板：完全封住气泡底部入口
  - 侧面封板：封住气泡下方 30% 区域的一侧
  - 随机在左侧或右侧留缺口通过
- **碰撞效果：** 弹回，速度 ×0.6–0.65

### 5.4 水母

- **分布区间：** 30–80%
- **出现概率：** 35%（生成在相邻气泡之间）
- **机制：** 碰撞后海龟弹回当前气泡，不消耗生命
- **视觉：** 半透明粉紫色水母体 + 摆动触须 + 上下浮动

### 5.5 水流区域

- **分布区间：** 40–75%
- **机制：** 飞行中经过时受到水平推力，交替左右方向
- **强度：** 0.3–0.8，推力 = 方向 × 强度 × 0.08
- **视觉：** 水平方向的流线粒子特效

### 5.6 墙壁

- **出现条件：** 进度 ≥ 48% 或已达第 24 个气泡
- **机制：** 屏幕两侧出现岩壁，碰撞后反弹（速度 ×0.82）
- **视觉：** 渐入显示，带砖纹理

---

## 6. 生命系统

### 6.1 生命值

- 生命次数由所选龟龟的稀有度决定：
  - C / B / A 级：10 次
  - S 级：12 次
  - SS / SSS 级：14 次
  - 测试龟（TEST）：无限（999）
- 每次坠落消耗 1 条生命

### 6.2 复活机制

- 生命 > 0 时自动复活，返回当前气泡
- 复活后获得 1.5 秒（90 帧）无敌闪烁
- 生命耗尽则 Game Over

---

## 7. 角色系统（龟龟选择）

### 7.1 角色来源

角色数据从主站 `petState`（localStorage）同步，基于宠物收集系统。

### 7.2 角色列表

| 稀有度 | 数量 | 颜色系 | 代表角色 |
|--------|------|--------|----------|
| C 级 | 3 | 绿色 | 小龟、石头龟、竹叶龟 |
| B 级 | 8 | 蓝色 | 天使龟、忍者龟、幽灵龟等 |
| A 级 | 8 | 深蓝 | 彩虹龟、海盗龟、闪电龟等 |
| S 级 | 6 | 紫色 | 凤凰龟、赛博龟、星际龟等 |
| SS 级 | 2 | 金色 | 缩头乌龟、无头龟 |
| SSS 级 | 1 | 红色 | 龟壳 |
| TEST | 1 | 紫色 | 测试龟（始终可用） |

### 7.3 角色差异

- 不同稀有度决定生命次数（见第 6 节）
- 各角色有独立 sprite 图片，支持回退到彩色圆形渲染
- 只有已拥有的龟龟可以选择

---

## 8. 计分与奖励

### 8.1 进度分数

- 分数 = 进度百分比 × 100（满分 10000）
- 通关固定 10000 分

### 8.2 龟币奖励

- 奖励 = `Math.floor(进度 / 5)` 龟币
- 通关固定 +50 龟币

### 8.3 跳级加分

- 跳过 1 个气泡：显示"跳级!"
- 跳过 2 个气泡：显示"跳级! x2!"
- 跳过 3+ 个气泡：显示"跳级! x3!"

---

## 9. 排行榜系统

### 9.1 本地排行榜

- 存储在 localStorage，默认预置 5 个 NPC 分数
- 保留 Top 10

### 9.2 在线排行榜（可选）

- 后端：Supabase（PostgreSQL）
- 按周计算（ISO 周），每周刷新
- 数据表字段：玩家名、分数、存活时间、最佳连击、周标识
- 玩家首次进入时输入昵称（2–12 字），持久化存储
- 在线不可用时自动降级为本地排行

---

## 10. 音效系统

### 10.1 技术方案

- 使用 Web Audio API 纯代码生成，零外部音频文件
- 全局音量开关，状态持久化到 localStorage

### 10.2 音效列表

| 音效 | 触发场景 | 波形 |
|------|----------|------|
| `sfxJump` | 跳跃 | sine，320→580Hz 上扬 |
| `sfxLand` | 着陆 | triangle，140→55Hz 下降 |
| `sfxBounce` | 墙壁/障碍反弹 | triangle，220→110Hz |
| `sfxNearMiss` | 近距离擦过 | sine，880→1320→660Hz |
| `sfxDeath` | 死亡 | sawtooth + 噪声 |
| `sfxCombo` | 连击 | sine，C D E F G A B 音阶递升 |
| `sfxPickup` | 拾取 | sine，三音阶上行 |
| `sfxShield` | 复活/护盾 | sine，440→880Hz |
| `sfxCountdown` | 倒计时 | sine 440Hz |
| `sfxGo` | 开始 | sine 三音阶 |
| `sfxMilestone` | 里程碑/通关 | sine C5-E5-G5-C6 琶音 |

### 10.3 背景音乐

- 纯代码生成的 Chiptune 风格循环 BGM
- BPM 130，2 个乐句循环
- 包含旋律（sine）、和声（triangle）、贝斯（square）、琶音（square）、底鼓、踩镲
- 淡入 0.5 秒启动，淡出 0.4 秒停止
- 暂停时淡出，恢复时淡入

## 11. 视觉表现

### 11.1 背景系统

三层背景图 + 动态渐变：

| 图层 | 文件 | 可见条件 | 视差速率 |
|------|------|----------|----------|
| 渐变底色 | 代码生成 | 始终 | — |
| 海中背景 | 海中bg.png | 始终 | 0.3× 相机速度 |
| 海面背景 | 海面bg.png | 进度 > 75% | 固定海面位置 |
| 海底背景 | 海底bg.png | 海底可见时 | 固定地面位置 |

### 11.2 视觉特效

- **光线：** 6 条从上方射下的光柱，随进度变亮，轻微摇摆
- **环境气泡：** 上升的小气泡，数量随深度增加
- **海面波浪：** 正弦波动画水面线
- **粒子系统：** 着陆爆发、碰撞火花、跳级彩色粒子
- **飞行拖尾：** 海龟颜色的渐隐轨迹
- **着陆动画：** 压扁拉伸效果（squash & stretch）
- **屏幕震动：** 碰撞、死亡时触发
- **无敌闪烁：** 复活后 1.5 秒闪烁
- **进度条脉冲：** 50%+ 进度条边框闪烁，颜色从橙色渐变到红色

### 11.3 颜色方案

| 变量 | 色值 | 用途 |
|------|------|------|
| `--bg` | #06111a | 深海底色 |
| `--g` | #06d6a0 | 绿色（进度、已达气泡） |
| `--b` | #4cc9f0 | 蓝色（下一个气泡、标题） |
| `--y` | #ffd93d | 黄色（角度指示器、金币） |
| `--r` | #ff6b6b | 红色（警告、生命） |
| `--p` | #c77dff | 紫色（S 级角色） |

## 13. 适配与响应式

### 13.1 布局

- 游戏区域最大宽度 420px，居中显示
- Canvas 铺满游戏容器
- 高度 = 视口高度 - 导航栏（45px）

### 13.2 触屏适配

- 触控目标最小 44px（暂停、音量按钮）
- 禁用页面缩放 (`user-scalable=no`)
- 禁用文本选择
- `touch-action: manipulation`

### 13.3 小屏适配（≤480px）

- 缩小导航栏字号
- 缩小 HUD 间距
- 隐藏排行榜面板
- 缩小角色选择器
- 缩小开始/结算界面元素
