# 龟龟对战 — 模块总结文档

## 项目概述

龟龟对战是一个 **3V3** 回合制策略战斗网页游戏，玩家选择3只龟组成队伍（前排2+后排1），与AI或其他玩家对战。

**技术栈**：纯前端 HTML/CSS/JS，PeerJS 实现 P2P 线上对战
**文件结构**：

## 游戏模式

| 模式 | 说明 |
|------|------|
| **野生对局 (PVE)** | 3v3 对抗AI控制的随机龟队，难度可选（简单/普通/困难） |
| **Boss挑战** | 3v1 挑战增强Boss（HP×3.5, ATK×1.3, DEF×1.5, 每回合2次行动） |
| **深海闯关** | 5关连续挑战，状态延续，每关间可重新站位，Boss关击败后获胜 |
| **线上对战 (PVP)** | P2P实时对战（房间制），PeerJS + TURN服务器 |
| **快速匹配** | 占位按钮（待后端支持） |

### 战场规则（每日随机）
| 规则 | 效果 |
|------|------|
| 🔥烈焰之日 | 所有伤害附带灼烧 |
| ⚡雷暴之日 | 全体暴击率+30% |
| 💀亡灵之日 | 所有龟首次死亡以15%HP复活 |
| 🛡️铁壁之日 | 护盾效果×2 |
| ⚔️狂暴之日 | 攻击力+40%，护甲-20% |
| 💧深海之日 | 魔抗+30%，魔法伤害-20% |
| 🎁装备之日 | 每3回合选1件装备 |
| 🎲正常对局 | 无额外规则 |

## 等级系统

- 等级范围：1-10级
- 每级+5%属性（ATK/HP/DEF/MR），10级=+45%
- 技能解锁：Lv.1可选index 0-2，Lv.4解锁index 3，Lv.7解锁index 4
- 等级存储在 localStorage petState.levels
- 在图鉴/宠物中心调整等级

---

## 技能配置系统

- 每只龟有5个技能（skillPool index 0-4）
- 战斗时携带3个（默认0,1,2），高等级可选更多
- 部分技能有互斥关系（conflictsWith）
- 被动技能（passiveSkill:true）战斗开始时自动应用，不占行动回合
- 组合技（2只特定龟同队触发，需双方Lv.9+）

## 核心机制

### 伤害系统
- **物理伤害**（红色）：受护甲减免，公式 `减免% = DEF/(DEF+40)`
- **魔法伤害**（蓝色）：受魔抗减免，公式 `减免% = MR/(MR+40)`
- **真实伤害**（白色）：无视所有防御
- **护盾吸收顺序**：泡泡护盾 → 普通护盾 → HP
- **负防御**：穿甲超过防御时，伤害被放大（最高×2）

### 增益/减益系统
- **数值型buff**（基于施法者属性）：攻击力+、护甲+、魔抗+
- **效果型debuff**（基于目标属性）：攻击力-%、护甲-%、魔抗-%
- **持续效果**：灼烧、中毒、诅咒、流血、治疗削减
- **特殊状态**：闪避、恐惧、眩晕、嘲讽、反弹、虫洞标记、墨迹标记、猎杀印记

### 站位系统
- 前排（最多3个）、后排（最多3个）
- 默认优先攻击前排，前排全灭才打后排
- 部分技能可无视站位（ignoreRow）或击至前排（knockToFront）
- 嘲讽buff强制被攻击

### 体型变化
- 龟的精灵图大小随最大生命值变化（0.9x ~ 1.15x）
- 获得最大生命值时变大，被偷取时缩小
- 0.4s平滑过渡动画

### 特殊机制
| 机制 | 龟 | 说明 |
|------|-----|------|
| 墨迹增伤 | 线条龟 | 全局生效，每层+5%受伤 |

| 星能充放 | 星际龟 | 造伤62%转星能，技能后释放30%，满能流星100% |
| 装备系统 | 宝箱龟 | 累积财宝值抽装备 |

| 多重打击 | 赌神龟 | 每段40%概率额外打击，连锁递减 |
| 结晶引爆 | 水晶龟 | 叠4层引爆10%目标HP魔法伤害 |

## 音效系统

- **Web Audio API buffer**：fetch→ArrayBuffer→decodeAudioData，播放延迟<5ms
- **sfxMap自动触发**：浮动数字按CSS class自动匹配音效（命中/暴击/破盾/闪避/护盾/治疗）
- **节流防重复**：_playSfxThrottled 50-100ms冷却，防止同帧多次播放
- **BGM系统**：菜单/战斗/Boss三首，静音状态持久化

| 名称 | 文件 | 用途 |

## 闯关模式（深海闯关）

- 5关制，难度递增（0.9/1.0/1.1/1.2/Boss）
- 状态延续：HP/护盾/buff在关卡间保留
- 关卡间可重新调整站位
- 每关可获得1件装备（18件装备池，分属性/特效两类）
- Boss关：HP×3.5, 每回合2次行动

### 装备池（18件）
- **属性类(8)**：海藻短刃(ATK+15%)、珊瑚护甲(DEF+20%)、深海贝壳(MR+20%)、生命珍珠(HP+60)、锋利鲨齿(暴击+20%)、重击锤(暴伤+25%)、穿甲珊瑚刺(穿甲+6)、灵能水晶(魔穿+6)
- **特效类(10)**：吸血海星(吸血12%)、荆棘海胆(反伤10%)、灼热火珊瑚(附带灼烧)、冰封水母(15%眩晕)、治愈海葵(HOT 5%HP/回合)、幽灵墨鱼(闪避+15%)、愤怒河豚(低血翻倍ATK)、坚韧龟壳(每段-5伤害)、连击章鱼爪(20%追加打击)、复活海螺(首次死亡20%HP复活)

## 回合流程

```
beginTurn()
  ├── 显示"第 N 回合"
  ├── processBuffs()
  │   ├── 每回合被动（+ATK、+HP、+DEF、棱镜、浮游炮、充能等）
  │   ├── 随从被动
  │   ├── DOT伤害（灼烧/中毒/诅咒/流血）
  │   └── buff倒计时 & 过期清理
  ├── 装备被动（HOT、低血暴怒）
  └── 进入行动阶段

nextSideAction()
  ├── 左方行动（玩家/AI）× 3只龟
  │   ├── 40秒计时器（超时自动出招）
  │   ├── 选龟 → 选技能 → 选目标
  │   └── executeAction() → 技能效果 → triggerOnHitEffects()
  ├── 右方行动
  └── finishSide()

finishSide() (双方行动完)
  ├── 随从回合末自动出招
  ├── 财神龟金币发放
  ├── 闪电龟回合电击
  ├── 龟壳储能波击
  ├── turnNum++ → beginTurn()
  └── 检查胜负
```

## PVP对战

- **P2P架构**：PeerJS + Metered TURN服务器
- **状态同步**：buildStateSync/applyStateSync（15+字段）
- **确定性RNG**：种子同步，双方计算结果一致
- **断线判负**：连接断开时对方自动获胜
- **房间制**：创建房间获取ID，对方输入ID加入
