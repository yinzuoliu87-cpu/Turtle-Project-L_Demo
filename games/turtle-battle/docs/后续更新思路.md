## v3.0 现状盘点

### 核心玩法 ✅
- 3v3回合制，前后排站位（6格阵型 + 嘲讽/前排优先/换位机制）
- 28只龟，120+技能，所有龟强度持平（稀有度≠战力）
- 等级系统（1-10级，每级+5%属性，Lv.5 解锁技能槽3，Lv.10 解锁技能槽4）
- 图鉴系统（书签式，可调等级+配置技能）
- 战场规则（8种每日随机）
- 12+组合技（双龟联动，需双方Lv.9+）
- 确定性 RNG PVP 同步

### 战斗动画系统（V3 新增）
- KOF 风格技能演出（镜头拉近 + 施法者冲锋 + 弹道/冲击波）
- 小龟 **龟派气波**：PixelLab 15 帧 sprite + WAAPI 物理仿真连击（重力驱动 3 连弹）
- 小龟 **过肩摔**：施法者冲过去抓取 + 目标抛物线翻转砸地 + PixelLab 9 帧冲击波 sprite
- 伤害浮字系统（按伤害量自动分档 20-42px，暴击 ×1.2）
- 伤害类型独立分色 + 垂直偏移避免重叠

### 模式 ✅
- PVE 3v3（三档难度）
- Boss挑战 3v1（HP×3.5，每回合2行动）
- 深海闯关（5关连续，血量延续，18件装备）
- P2P线上对战（PeerJS + TURN）

### 系统 ✅
- Web Audio API 音效（7个真实音效 + BGM 3首）
- 体型变化动画（HP 变化时 0.9x~1.15x）
- 龟币经济
- 移动端适配
- 新手引导

### 工程 ✅
- engine.js 拆分 7 文件 + skills/ 按龟分文件
- Playwright 自动化测试
- localStorage 持久化

**以小龟和 PixelLab 生成的 VFX 为基准**：
- Chi-wave (15 帧 128×128)：~35 credits
- Slam-impact (9 帧 128×128)：~20 credits

**推到 28 只龟**：

| 资产 | 每只 credit | 总 credits |
|---|---|---|

## AI + 0 外包能做到什么（v3 更新后）

**AI 能做好**：
- 龟 / UI / 装备 / 背景像素图（PixelLab、Midjourney）
- 多帧动画（PixelLab 的 animated generator 效果不错）
- 所有代码逻辑（Claude 写）
- 概念设计

**AI 做不好，需手动**：
- 跨帧一致性（需反复抽卡 + 手动挑选）
- UI 精确像素对齐
- 版权干净的 BGM / 高质量语音

**结论**：v3.0 → v3.4 都可以 **AI + 0 外包** 完成，总成本 $30-50（PixelLab）。v4.0 若要品质跃升再投钱。
