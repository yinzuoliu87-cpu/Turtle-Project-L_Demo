# 龟龟对战 — 后续更新思路

## 一、核心玩法升级

### 1.1 3v3 + 自由站位

当前2v2改为3v3，增加前排/后排概念，**站位完全由玩家自由选择，不根据龟的类型限制**：

**站位规则**：
- 选完3只龟后，进入站位阶段，玩家自由拖拽放置前排/后排
- 可选阵型：3前0后、2前1后、1前2后、0前3后
- 任何龟都能放任何位置，策略完全由玩家决定

**前后排机制**：
- **前排**：敌方单体技能优先选中前排目标（前排全灭前，后排不会被单体选中）
- **后排**：不会被敌方单体技能直接选中，但受到AOE伤害正常
- **站位切换**：战斗中可以消耗一个行动机会换位（前→后或后→前）

**策略示例**：
- 3前0后：全压制，高防御阵容硬刚
- 1前2后：一只坦克扛伤害，两只后排安全输出
- 0前3后：全后排无前排保护，但对方单体技能强制随机选目标
- 2前1后：平衡阵型，灵活应对

### 1.2 技能树 / 技能装配

每只龟拥有5-6个可选技能，战前选择装配3个：

- **技能槽**：每只龟3个技能槽（与现在相同）
- **技能池**：每只龟拥有5-6个技能（比现在多2-3个）
- **战前装配**：选龟后进入技能装配界面，从技能池中选3个上场
- **策略深度**：同一只龟可以走攻击流、防御流、辅助流

**示例 — 凤凰龟技能池**：
| 技能 | 类型 | 说明 |
|------|------|------|
| 灼烧 | 进攻 | 现有：90%ATK魔法+灼烧 |
| 熔岩盾 | 防御 | 现有：75%ATK护盾+反击 |
| 烫伤 | 控制 | 现有：破盾+削弱+灼烧+治疗削减 |
| 涅槃之焰 | 进攻 | **新增**：消耗30%当前HP，对全体敌人造成大量火焰伤害 |
| 火焰净化 | 辅助 | **新增**：清除友方所有减益，每清一个回复10%HP |

### 1.3 回合间装备选择

每隔N回合，双方从随机装备池中选择装备：

- **触发时机**：每3回合（或击杀敌方龟时）
- **选择方式**：从3件随机装备中选1件，装给任意己方龟
- **装备效果**：攻击力+X、护甲+X、生命偷取+X%、技能CD-1等
- **装备上限**：每只龟最多装2件
- **参考**：宝箱龟现有的装备系统可以作为基础，扩展为通用系统

### 1.4 PvE闯关模式

连续挑战5-10关，龟龟血量延续，每关之间有选择：

- **关卡结构**：5关普通怪 + 1关Boss = 1个章节
- **血量延续**：每关结束后HP不回满，只回复20-30%
- **关间选择**：每关之间从3个选项中选1个（回复HP / 获得装备 / 临时增益）
- **难度递增**：每关敌人属性略微提升
- **奖励递增**：打到越后面龟币奖励越多
- **死亡惩罚**：龟阵亡后本次闯关不可再用，需要用剩余龟继续
- **阵容深度**：需要准备5-6只龟而非2只，增加龟的使用率

### 1.5 每局随机规则

每场战斗随机附加一条特殊规则：

- 🔥 **烈焰之日**：所有伤害附带灼烧效果
- ❄️ **冰封之日**：所有龟速度降低50%，但护甲+30%
- ⚡ **雷暴之日**：暴击率全体+30%，暴击伤害翻倍
- 💀 **亡灵之日**：所有龟首次死亡后以50%HP复活一次
- 🎲 **混沌之日**：每回合技能CD随机重置
- 🛡️ **铁壁之日**：所有护盾效果+100%
- 💧 **治愈之日**：所有回复效果+50%
- ⚔️ **狂暴之日**：所有攻击力+40%，但护甲-20%

**显示方式**：战斗开始前弹出本局规则，双方都能看到。

### 1.6 连携技

特定龟组合可以释放合体技能：

- **触发条件**：两只特定龟同时存活，消耗双方各1个行动机会
- **连携按钮**：当条件满足时，技能面板出现特殊的连携技按钮
- **伤害加成**：连携技伤害高于两只龟分别行动的总和

**连携技示例**：
| 组合 | 连携技名 | 效果 |
|------|---------|------|
| 熔岩龟 + 凤凰龟 | 烈焰风暴 | 全体敌人200%ATK火焰伤害+灼烧3回合 |
| 寒冰龟 + 水晶龟 | 绝对零度 | 全体敌人冻结1回合（跳过行动）+魔抗-30% |
| 闪电龟 + 赛博龟 | 电磁脉冲 | 清除全体敌人护盾+150%ATK真实伤害 |
| 猎人龟 + 忍者龟 | 暗影突袭 | 对血量最低敌人造成300%ATK真实伤害 |
| 石头龟 + 钻石龟 | 磐石壁垒 | 全队获得200%DEF护盾+反伤3回合 |
| 天使龟 + 幽灵龟 | 审判降临 | 全体敌人当前HP 20%魔法伤害+诅咒2回合 |
| 糖果龟 + 泡泡龟 | 甜蜜陷阱 | 偷取全体敌人10%最大HP+储存为泡泡值 |

## 二、成长系统

### 2.1 龟龟等级（1-10级）

- **升级方式**：使用某只龟完成对局获得经验值
- **等级效果**：每级基础属性小幅提升（+2% HP/ATK/DEF/MR）
- **技能解锁**：达到特定等级解锁新的可选技能（配合技能装配系统）
- **外观变化**：5级和10级时龟的外观/精灵图升级
- **平衡考量**：PvP匹配时根据龟等级加权，避免高等级碾压

### 2.2 图鉴系统

- **收集进度**：显示已解锁/未解锁的龟
- **解锁条件**：完成特定任务解锁新龟（如"用C级龟赢20场解锁XX龟"）
- **详细信息**：每只龟的胜率、使用次数、最佳搭档、最佳战绩
- **成就徽章**：每只龟有独立成就（如"猎人龟：单局斩杀3只敌人"）

### 2.3 赛季任务

- **每日任务**：完成3个日常任务获得龟币（如"赢1场PvE"、"使用B级龟出战"）
- **周任务**：完成获得稀有奖励（如"用10只不同的龟赢过"）
- **赛季挑战**：长期目标（如"达到10连胜"、"闯关模式通关5次"）

## 三、画风与动效提升

### 3.1 画风方向参考

| 游戏 | 画风特点 | 可借鉴 |
|------|---------|--------|
| 云顶之弈 | 3D Q版模型，流畅的自动战斗动画 | 技能释放时的镜头特写 |
| 怪物圣所 | 像素精灵风，精致的帧动画 | 每只龟的待机/攻击/受伤/死亡动画 |
| Dicey Dungeons | 矢量插画风，干净明快的色彩 | UI的色彩搭配和卡片设计感 |
| 破败之王 | 手绘2D，暗色调+高光特效 | 技能释放的粒子特效 |

**龟龟对战建议方向**：
- **像素精灵风**（已有基础）→ 继续精化
- 每只龟需要至少4套动画帧：**待机**、**攻击**、**受伤**、**技能释放**
- 当前只有待机动画，需要补充攻击/受伤帧

### 3.2 战斗动画提升

**当前问题**：技能释放只有数字跳动+卡片抖动，缺少视觉冲击感

**改进方向**：

1. **技能特效**
   - 每个技能有独立的CSS/Canvas动画
   - 物理技能：刀光/拳影划过屏幕
   - 魔法技能：光球/火焰/冰晶飞向目标
   - 真实伤害：金色剑气穿透效果
   - 治疗/护盾：绿色/白色光圈上升

2. **受击反馈**
   - 被攻击时整个卡片闪红（物理）/闪蓝（魔法）/闪金（真实）
   - 屏幕边缘泛红/震动（大伤害时）
   - 暴击时屏幕短暂放大+闪白

3. **镜头效果**
   - 大招释放时背景变暗，聚焦释放者
   - 击杀时慢镜头效果（0.5秒放慢）
   - 变身（熔岩龟）时全屏特效

4. **死亡/复活动画**
   - 死亡：龟的精灵图碎裂/淡出
   - 凤凰复活：火焰从灰烬中升起
   - 无头龟锁血：暗紫色光环闪烁

### 3.3 音效提升

**当前状态**：用Web Audio API合成的简易音效

**改进方向**：

1. **BGM**
   - 主菜单：轻松的像素风BGM
   - 战斗中：紧张的战斗BGM（可随回合加速）
   - Boss战：专属Boss BGM
   - 胜利/失败：独立结算音乐

2. **技能音效**
   - 每个伤害类型独立音效（物理砍击/魔法爆裂/真实穿刺）
   - 护盾音效（展开/吸收/破碎）
   - 治疗音效（清脆的叮咚声）
   - 暴击音效（比普通攻击更厚重）

3. **环境音效**
   - 回合开始提示音
   - 倒计时最后10秒心跳音效
   - 连击/多段攻击的节奏音效

4. **语音（远期）**
   - 每只龟有简单的语音（如"呀！""嘿！"）
   - 可以用TTS或请声优录制

### 3.4 UI视觉提升

1. **卡片设计**
   - 稀有度边框用金属质感渐变（而非纯色边框）
   - S级以上卡片有呼吸光效/粒子背景
   - 被动图标加发光/脉动效果

2. **技能面板**
   - 技能按钮有类型色标（物理红/魔法蓝/辅助绿）
   - CD恢复动画（扇形遮罩逐渐消失）
   - 可用技能轻微脉动提示

3. **血条设计**
   - 分段血条（每段=一定HP，掉一段有视觉反馈）
   - 护盾条有微光流动效果
   - 受伤时血条闪烁

## 六、画风确认：2D像素风

### 6.1 确认方向

**画风：2D像素风 + 2.5D斜视角战斗场景（海洋主题）**

- 角色精灵图：2D侧面像素画（现有资源兼容，不需要多角度）
- 战斗背景：2.5D等距斜45°俯视角（有地面纵深感，角色"站"在场景里）
- UI/图标：2D像素风

视角参考游戏：
- **Sea of Stars** — 海洋主题最贴合，像素风+动态光影
- **怪物圣所 (Monster Sanctuary)** — 3v3站位和战斗动画
- **Octopath Traveler** — 像素风做出华丽感的标杆

选择理由：
- 已有28只龟的像素精灵图，切换其他风格等于全部作废
- 2.5D斜视角背景让场景有纵深感，但角色仍用2D侧面图不需要重画
- AI画图（DALL-E/Midjourney）在像素风上最成熟，可持续产出
- Web端2D性能稳定，手机浏览器无压力

### 6.2 视觉标识

**"海洋像素风"** — 以海洋/海底为主题的像素艺术风格
- 背景：珊瑚礁、海底洞穴、沉船等场景
- 色调：深蓝+青绿的海洋色系为底，龟龟用鲜艳色彩突出
- UI元素：贝壳按钮、珊瑚边框、气泡装饰
- 特效：水波纹、气泡粒子、光线折射

### 6.3 跨端适配策略：PC横版 + 手机竖版

**一套资源，两套布局**：

| 设备 | 方向 | 战斗布局 |
|------|------|---------|
| PC (>768px) | 横版16:9 | 左3只 ← VS → 右3只，技能栏底部一行展开 |
| 手机 (<768px) | 竖版9:16 | 上3只(敌方) VS 下3只(我方)，技能栏底部紧凑按钮 |

**精灵图统一规格**：
- 龟精灵图：128×128 像素画（含动画帧）→ PC显示128px，手机显示64px
- 被动/装备图标：48×48 像素画 → PC显示32px，手机显示24px
- 战斗背景：一张大图（960×640），PC横向铺满，手机裁剪中心区域
- UI元素（按钮/边框/面板）：9-patch切片，任意尺寸拉伸不变形
- 全局 `image-rendering: pixelated` 确保像素风缩放不模糊

**CSS响应式切换**，不需要两套代码：
```
PC横版：
┌──────────────────────────────────┐
│         战斗背景（横版）           │
│  [龟1]                   [龟1]   │
│  [龟2]      VS           [龟2]   │
│  [龟3]                   [龟3]   │
├──────────────────────────────────┤
│  [技能1] [技能2] [技能3]  [日志]  │
└──────────────────────────────────┘

手机竖版：
┌────────────┐
│  战斗背景    │
│ [龟1][龟2][龟3] ← 敌方
│     VS      │
│ [龟1][龟2][龟3] ← 我方
├────────────┤
│ [技能][技能] │
│ [技能] [📜📊]│
└────────────┘
```

### 6.4 2D高级感实现方式

**场景纵深（CSS视差）**：
```
背景层：远处的珊瑚/气泡（慢速视差滚动）
中景层：战斗场地（龟站在这里）
前景层：近处的海草/光影（轻微遮挡）
```
三层叠加产生深度感，不需要3D引擎。

**动态光影**：
- 龟身上根据技能类型投射不同颜色的光（火=橙红、冰=冰蓝、电=金黄）
- CSS `filter: drop-shadow` + `mix-blend-mode` 实现

**粒子系统**：
- 水中气泡持续上浮（Canvas 2D粒子）
- 技能释放时的光粒子爆发
- 环境光点漂浮

**流畅动画**：
- 攻击：龟冲刺到对面→回弹（CSS transform）
- 技能弹道：光球/火焰/冰锥从左飞到右
- 受击：全屏闪光+屏幕震动
- 暴击：慢镜头0.3秒+数字放大

## 七、战斗场景架构演进

### 7.1 当前问题

现在是"卡片式"DOM布局——龟在各自的卡片框里，没有统一的战斗场景。和怪物圣所等游戏对比，差距在于：
- 怪物圣所：角色站在场景**里面**，有纵深和空间感
- 龟龟对战：角色在卡片**上面**，看起来像网页应用而非游戏

### 7.2 怪物圣所的战斗画面结构

```
┌─────────────────────────────────────┐
│           战斗背景（珊瑚/洞穴）        │
│                                     │
│   [龟3]                    [龟3]    │
│         [龟2]        [龟2]          │
│   [龟1]                    [龟1]    │
│                                     │
│  ═══════ 地面/水面线 ═══════         │
└─────────────────────────────────────┘
           技能面板 / 状态栏
```

左边3只面朝右，右边3只面朝左，有前后排纵深感。HP条浮在角色头顶，技能特效在场景内飞行。

### 7.3 分阶段演进方案

**第一阶段（近期，纯CSS改动）**：
- 保持现有DOM卡片布局
- 给战斗区域加一个AI生成的背景图（海底珊瑚礁）
- 卡片改为半透明，让背景透出来
- 视觉立刻从"网页应用"变成"有场景的游戏"

**第二阶段（中期，DOM重构）**：
- 战斗区域改为一个大容器，龟的精灵图直接站在上面
- HP条浮在龟头顶（absolute定位）
- 属性/状态信息折叠到底部面板
- 攻击时龟的精灵图移动（CSS transform冲向对方→回弹）

**第三阶段（远期，Canvas渲染）**：

## 十、主站联动与社交系统

龟投主站是热点预测/竞猜平台，龟币是核心经济货币。小游戏需要深度嵌入这个生态。

### 10.0 龟币的社交价值

龟投的核心是投票预测市场，龟币是虚拟货币。虽然没有实际价值，但参考B站硬币/等级体系，虚拟货币的社交地位感是留存的核心驱动力：

**为什么用户会在意虚拟币**：
- **稀缺性**：龟币获取有限（每日首胜、竞猜、小游戏），必须做选择怎么花
- **社交展示**：龟币余额、等级、皮肤、战绩是"身份标识"，别人能看到
- **社区认同**：投票预测正确 = 眼光好 = 社区声望
- **沉没成本**：用户攒了越多龟币/皮肤/等级，越不愿意离开

**龟币生态设计原则**：
- **赚取渠道多样**：竞猜预测、小游戏、每日签到、赛季奖励、社交互动
- **消费场景丰富**：竞猜下注、对战赌注、皮肤购买、喂食龟龟、解锁龟
- **展示无处不在**：个人主页龟币余额、论坛帖子旁的等级徽章、对战结果分享卡上的皮肤
- **适度通缩**：竞猜亏损、对战门票等消耗渠道，防止龟币贬值
- **永远不卖龟币**：保持"只能靠玩赚"的稀缺性，付费只卖皮肤/装饰

### 10.1 龟币经济打通

小游戏与主站共享同一套龟币系统：
- 小游戏赚的龟币直接用于主站竞猜下注
- 主站竞猜赢的龟币可用于小游戏解锁龟/皮肤/装备
- 形成核心循环：**玩游戏赚币 → 竞猜赢更多 → 解锁更强的龟 → 赢更多**

### 10.2 宠物养成 → 战斗加成（参考怪物圣所）

主站宠物中心的养成系统直接影响战斗属性：

**喂食系统**：
| 食物 | 获取方式 | 加成效果 |
|------|---------|---------|
| 海藻 | 每日签到 | HP +2%（持续1天） |
| 珊瑚果 | 竞猜胜利奖励 | ATK +3%（持续1天） |
| 深海珍珠 | 稀有掉落/商店 | 全属性 +2%（持续3天） |
| 黄金鱼卵 | 赛季奖励 | 全属性 +5%（持续7天） |

**好感度等级**（参考怪物圣所的羁绊系统）：
| 好感度 | 需要互动次数 | 解锁效果 |
|--------|------------|---------|
| ❤ 1级 | 10次喂食/互动 | 解锁被动描述 |
| ❤❤ 2级 | 30次 | 基础属性 +3% |
| ❤❤❤ 3级 | 80次 | 解锁额外可选技能 |
| ❤❤❤❤ 4级 | 150次 | 基础属性 +6%，解锁专属皮肤 |
| ❤❤❤❤❤ 5级 | 300次 | 基础属性 +10%，解锁专属连携技 |

**互动方式**：
- 喂食（消耗食物道具）
- 抚摸/玩耍（每日次数限制）
- 一起战斗（每场对战+1互动）
- 出海冒险（龟龟出海通关+2互动）

### 10.3 社交对战系统

**论坛/开战广场挑战**：
- 在主站论坛帖子下方可以直接发起"龟龟对战"挑战
- 格式："我用[彩虹龟+闪电龟]挑战你，赌注100龟币！"
- 对方点击接受 → 自动跳转对战界面 → 结果自动回帖
- 围观群众可以押注哪方赢

**社交赌注规则**：
| 赌注类型 | 说明 |
|---------|------|
| 普通赌注 | 双方各押X龟币，赢家拿走全部 |
| 挑战赌注 | "用指定的龟赢我"，赔率更高 |
| 围观押注 | 旁观者可以押注某一方赢 |
| 排行赌注 | 排行榜前10名之间的对战自动开放押注 |

**对战结果分享**：
- 对战结束自动生成分享卡片（精美截图+胜负+数据）
- 一键分享到主站论坛/微信/QQ
- 包含：双方阵容、回合数、MVP龟（伤害最高）、精彩操作回放

### 10.4 皮肤系统

**皮肤获取方式**：
| 途径 | 说明 |
|------|------|
| 好感度解锁 | 好感度4级解锁专属皮肤 |
| 赛季奖励 | 赛季排名前X%获得限定皮肤 |
| 龟币商店 | 用龟币购买 |
| 竞猜奖励 | 主站热点预测连续正确X次解锁 |
| 成就解锁 | 完成特定成就（如"10连胜"） |
| 限时活动 | 热点事件期间限定皮肤 |

**皮肤内容**：每款皮肤包含新立绘/换色，部分皮肤附带技能特效变色或专属音效。

**皮肤不影响属性**：纯装饰性，不提供任何战斗加成，保持PvP公平。

### 10.5 龟龟出海联动

- 出海成绩影响龟的成长经验值（到达越高层经验越多）
- 出海过程中随机获得对战用的一次性道具（如"战斗前全队回复20%HP"）
- 出海排行榜每周结算，前3名获得稀有皮肤碎片
- 出海途中遇到"漂流瓶"随机事件 → 获得龟币/食物/皮肤碎片

### 10.6 热点联动玩法

主站热点事件与小游戏联动：

**赛季制**：
- 每个大热点事件 = 一个赛季（如"2026世界杯赛季"）
- 赛季期间特定龟获得属性加成（"世界杯赛季：寒冰龟攻击力+10%"）
- 赛季结束结算排名，发放赛季专属奖励

**限时规则**：
- 热点事件触发限时对战规则（"AI大战：赛博龟+星际龟属性翻倍"）
- 限时Boss挑战（"世界杯Boss：⚽足球龟"特殊Boss）

**预测奖励**：
- 主站预测正确3次 → 解锁对战用的临时增益道具
- 主站预测连对5次 → 解锁限定皮肤
- 竞猜MVP（当周最多正确预测）→ 专属称号+头像框
