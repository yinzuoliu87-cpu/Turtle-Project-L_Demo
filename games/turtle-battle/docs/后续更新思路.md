# 龟龟对战 — 后续更新思路

## 一、核心玩法升级

### 1.1 3v3 + 前后排站位

当前2v2改为3v3，增加前排/后排概念：

- **前排**（1-2只龟）：承受单体攻击优先目标，适合坦克型龟
- **后排**（1-2只龟）：受到AOE伤害但不会被单体技能直接选中（除非前排全灭）
- **站位切换**：消耗一个行动机会换位，带来策略博弈
- **前排加成**：前排龟获得少量护甲/魔抗加成
- **后排加成**：后排龟获得少量攻击力/暴击加成

**对现有龟的影响**：
- 石头龟、钻石龟、水晶龟 → 天然前排（高防御）
- 猎人龟、闪电龟、星际龟 → 天然后排（高输出低防御）
- 双头龟 → 远程形态后排、近战形态前排
- 缩头乌龟 → 前排保护随从

### 1.2 技能树 / 技能装配

每只龟拥有5-6个可选技能，战前选择装配3个：

- **技能槽**：每只龟3个技能槽（与现在相同）
- **技能池**：每只龟拥有5-6个技能（比现在多2-3个）
- **战前装配**：选龟后进入技能装配界面，从技能池中选3个上场
- **策略深度**：同一只龟可以走攻击流、防御流、辅助流

**示例 — 凤凰龟技能池**：
| 技能 | 类型 | 说明 |
|------|------|------|
| 灼烧 | 进攻 | 现有：90%ATK魔法+灼烧 |
| 熔岩盾 | 防御 | 现有：75%ATK护盾+反击 |
| 烫伤 | 控制 | 现有：破盾+削弱+灼烧+治疗削减 |
| 涅槃之焰 | 进攻 | **新增**：消耗30%当前HP，对全体敌人造成大量火焰伤害 |
| 火焰净化 | 辅助 | **新增**：清除友方所有减益，每清一个回复10%HP |

### 1.3 回合间装备选择

每隔N回合，双方从随机装备池中选择装备：

- **触发时机**：每3回合（或击杀敌方龟时）
- **选择方式**：从3件随机装备中选1件，装给任意己方龟
- **装备效果**：攻击力+X、护甲+X、生命偷取+X%、技能CD-1等
- **装备上限**：每只龟最多装2件
- **参考**：宝箱龟现有的装备系统可以作为基础，扩展为通用系统

### 1.4 PvE闯关模式

连续挑战5-10关，龟龟血量延续，每关之间有选择：

- **关卡结构**：5关普通怪 + 1关Boss = 1个章节
- **血量延续**：每关结束后HP不回满，只回复20-30%
- **关间选择**：每关之间从3个选项中选1个（回复HP / 获得装备 / 临时增益）
- **难度递增**：每关敌人属性略微提升
- **奖励递增**：打到越后面龟币奖励越多
- **死亡惩罚**：龟阵亡后本次闯关不可再用，需要用剩余龟继续
- **阵容深度**：需要准备5-6只龟而非2只，增加龟的使用率

### 1.5 每局随机规则

每场战斗随机附加一条特殊规则：

- 🔥 **烈焰之日**：所有伤害附带灼烧效果
- ❄️ **冰封之日**：所有龟速度降低50%，但护甲+30%
- ⚡ **雷暴之日**：暴击率全体+30%，暴击伤害翻倍
- 💀 **亡灵之日**：所有龟首次死亡后以50%HP复活一次
- 🎲 **混沌之日**：每回合技能CD随机重置
- 🛡️ **铁壁之日**：所有护盾效果+100%
- 💧 **治愈之日**：所有回复效果+50%
- ⚔️ **狂暴之日**：所有攻击力+40%，但护甲-20%

**显示方式**：战斗开始前弹出本局规则，双方都能看到。

### 1.6 连携技

特定龟组合可以释放合体技能：

- **触发条件**：两只特定龟同时存活，消耗双方各1个行动机会
- **连携按钮**：当条件满足时，技能面板出现特殊的连携技按钮
- **伤害加成**：连携技伤害高于两只龟分别行动的总和

**连携技示例**：
| 组合 | 连携技名 | 效果 |
|------|---------|------|
| 熔岩龟 + 凤凰龟 | 烈焰风暴 | 全体敌人200%ATK火焰伤害+灼烧3回合 |
| 寒冰龟 + 水晶龟 | 绝对零度 | 全体敌人冻结1回合（跳过行动）+魔抗-30% |
| 闪电龟 + 赛博龟 | 电磁脉冲 | 清除全体敌人护盾+150%ATK真实伤害 |
| 猎人龟 + 忍者龟 | 暗影突袭 | 对血量最低敌人造成300%ATK真实伤害 |
| 石头龟 + 钻石龟 | 磐石壁垒 | 全队获得200%DEF护盾+反伤3回合 |
| 天使龟 + 幽灵龟 | 审判降临 | 全体敌人当前HP 20%魔法伤害+诅咒2回合 |
| 糖果龟 + 泡泡龟 | 甜蜜陷阱 | 偷取全体敌人10%最大HP+储存为泡泡值 |

## 二、成长系统

### 2.1 龟龟等级（1-10级）

- **升级方式**：使用某只龟完成对局获得经验值
- **等级效果**：每级基础属性小幅提升（+2% HP/ATK/DEF/MR）
- **技能解锁**：达到特定等级解锁新的可选技能（配合技能装配系统）
- **外观变化**：5级和10级时龟的外观/精灵图升级
- **平衡考量**：PvP匹配时根据龟等级加权，避免高等级碾压

### 2.2 图鉴系统

- **收集进度**：显示已解锁/未解锁的龟
- **解锁条件**：完成特定任务解锁新龟（如"用C级龟赢20场解锁XX龟"）
- **详细信息**：每只龟的胜率、使用次数、最佳搭档、最佳战绩
- **成就徽章**：每只龟有独立成就（如"猎人龟：单局斩杀3只敌人"）

### 2.3 赛季任务

- **每日任务**：完成3个日常任务获得龟币（如"赢1场PvE"、"使用B级龟出战"）
- **周任务**：完成获得稀有奖励（如"用10只不同的龟赢过"）
- **赛季挑战**：长期目标（如"达到10连胜"、"闯关模式通关5次"）

## 三、画风与动效提升

### 3.1 画风方向参考

| 游戏 | 画风特点 | 可借鉴 |
|------|---------|--------|
| 云顶之弈 | 3D Q版模型，流畅的自动战斗动画 | 技能释放时的镜头特写 |
| 怪物圣所 | 像素精灵风，精致的帧动画 | 每只龟的待机/攻击/受伤/死亡动画 |
| Dicey Dungeons | 矢量插画风，干净明快的色彩 | UI的色彩搭配和卡片设计感 |
| 破败之王 | 手绘2D，暗色调+高光特效 | 技能释放的粒子特效 |

**龟龟对战建议方向**：
- **像素精灵风**（已有基础）→ 继续精化
- 每只龟需要至少4套动画帧：**待机**、**攻击**、**受伤**、**技能释放**
- 当前只有待机动画，需要补充攻击/受伤帧

### 3.2 战斗动画提升

**当前问题**：技能释放只有数字跳动+卡片抖动，缺少视觉冲击感

**改进方向**：

1. **技能特效**
   - 每个技能有独立的CSS/Canvas动画
   - 物理技能：刀光/拳影划过屏幕
   - 魔法技能：光球/火焰/冰晶飞向目标
   - 真实伤害：金色剑气穿透效果
   - 治疗/护盾：绿色/白色光圈上升

2. **受击反馈**
   - 被攻击时整个卡片闪红（物理）/闪蓝（魔法）/闪金（真实）
   - 屏幕边缘泛红/震动（大伤害时）
   - 暴击时屏幕短暂放大+闪白

3. **镜头效果**
   - 大招释放时背景变暗，聚焦释放者
   - 击杀时慢镜头效果（0.5秒放慢）
   - 变身（熔岩龟）时全屏特效

4. **死亡/复活动画**
   - 死亡：龟的精灵图碎裂/淡出
   - 凤凰复活：火焰从灰烬中升起
   - 无头龟锁血：暗紫色光环闪烁

### 3.3 音效提升

**当前状态**：用Web Audio API合成的简易音效

**改进方向**：

1. **BGM**
   - 主菜单：轻松的像素风BGM
   - 战斗中：紧张的战斗BGM（可随回合加速）
   - Boss战：专属Boss BGM
   - 胜利/失败：独立结算音乐

2. **技能音效**
   - 每个伤害类型独立音效（物理砍击/魔法爆裂/真实穿刺）
   - 护盾音效（展开/吸收/破碎）
   - 治疗音效（清脆的叮咚声）
   - 暴击音效（比普通攻击更厚重）

3. **环境音效**
   - 回合开始提示音
   - 倒计时最后10秒心跳音效
   - 连击/多段攻击的节奏音效

4. **语音（远期）**
   - 每只龟有简单的语音（如"呀！""嘿！"）
   - 可以用TTS或请声优录制

### 3.4 UI视觉提升

1. **卡片设计**
   - 稀有度边框用金属质感渐变（而非纯色边框）
   - S级以上卡片有呼吸光效/粒子背景
   - 被动图标加发光/脉动效果

2. **技能面板**
   - 技能按钮有类型色标（物理红/魔法蓝/辅助绿）
   - CD恢复动画（扇形遮罩逐渐消失）
   - 可用技能轻微脉动提示

3. **血条设计**
   - 分段血条（每段=一定HP，掉一段有视觉反馈）
   - 护盾条有微光流动效果
   - 受伤时血条闪烁

## 六、画风独特性思考

龟龟对战需要形成自己的视觉标识，而不是模仿某一款游戏。建议方向：

**"海洋像素风"** — 以海洋/海底为主题的像素艺术风格
- 背景：珊瑚礁、海底洞穴、沉船等场景
- 色调：深蓝+青绿的海洋色系为底，龟龟用鲜艳色彩突出
- UI元素：贝壳按钮、珊瑚边框、气泡装饰
- 特效：水波纹、气泡粒子、光线折射

这种风格与"龟"的主题天然契合，且在市场上没有同类竞品使用。
