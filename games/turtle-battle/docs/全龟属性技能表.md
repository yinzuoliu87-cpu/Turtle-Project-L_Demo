# 全龟属性技能表

> 基础属性即为实战属性，无稀有度系数加成。
>
> 防御减伤公式：reduction% = DEF/(DEF+40)
>
> 穿甲：固定穿甲(armorPen)无视X点防御 + 百分比穿甲(armorPenPct)无视X%防御
>
> 稀有度系数：C=1.00 B=1.03 A=1.06 S=1.09 SS=1.12 SSS=1.15
