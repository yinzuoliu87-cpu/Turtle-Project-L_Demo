# 龟龟对战 — 模块需求文档

## 1. 模块概述

2V2 回合制战斗模块，玩家从已拥有的龟中选择 2 只组队，与 AI 或其他玩家进行战斗。

**入口**: 主页侧栏 → `games/turtle-battle/index.html`

---

## 2. 游戏模式

| 模式 | 说明 |
|------|------|
| 野生对局 (PvE) | 玩家选 2 只龟，AI 随机选 2 只对战 |
| 线上对战 | 创建/加入 6 位房间号，BroadcastChannel 通信（同浏览器） |

## 3. 属性系统

### 3.1 基础属性

| 属性 | 说明 |
|------|------|
| HP | 生命值，归零即阵亡 |
| ATK | 攻击力，技能基础伤害来源 |
| DEF | 防御力，减伤公式：`减伤% = DEF/(DEF+40)` |
| Crit | 暴击率 (5%~50%)，暴击 = 1.5× 伤害（可被爆伤加成提高） |
| ArmorPen | 穿甲值，计算减伤时无视目标对应点数的防御 |

### 3.2 减伤公式

```
减伤% = effectiveDEF / (effectiveDEF + 40)
effectiveDEF = 目标DEF - 攻击者穿甲值（最低0）
```

| DEF | 减伤% | 说明 |
|-----|-------|------|
| 5 | 11% | 低防 |
| 10 | 20% | |
| 15 | 27% | |
| 20 | 33% | 中防 |
| 30 | 43% | |
| 40 | 50% | 高防 |
| 60 | 60% | 极高 |

递减曲线，永远低于 100%。

### 3.3 稀有度体系

每升一级稀有度，基础属性 ×1.03（即 +3%）。

| 稀有度 | 属性倍率 | 颜色 |
|--------|---------|------|
| C | ×1.00 | #06d6a0 绿 |
| B | ×1.03 | #4cc9f0 蓝 |
| A | ×1.06 | #3a9abf 深蓝 |
| S | ×1.09 | #c77dff 紫 |
| SS | ×1.12 | #ffd93d 金 |
| SSS | ×1.15 | #ff6b6b 红 |

### 3.4 龟龟总数

共 28 只龟 (C×3, B×8, A×8, S×6, SS×2, SSS×1)。

## 4. 伤害计算

### 4.1 完整伤害流程

```
技能基础威力 basePower
  ├── 可含 atkScale (×攻击力)
  ├── 可含 defScale (×防御力)
  └── 可含 hpPct   (+%目标最大HP)

若多段命中(hits>N)且使用系数，总量均分到每段

每段计算:
  1. 闪避判定 → 闪避则跳过
  2. 暴击判定 → critMult = 1.5 + 额外爆伤加成
  3. 普通部分 = (basePower - pierce) × critMult × (1 - 减伤%)
  4. 穿透部分 = pierce × critMult  (无视DEF)
  5. 总伤害 = 普通 + 穿透
  6. 吸收: 泡泡盾 → 普通护盾 → HP
  7. 触发被攻击被动 (反伤/泡泡储存/夹子/熔岩盾反击等)
```

### 4.2 伤害颜色编码

| 类型 | 颜色 | 说明 |
|------|------|------|
| 直接伤害 | #ff4444 红 | 经过DEF减免的普通伤害 |
| 穿透伤害 | #d06bff 紫 | 无视防御，打护盾 |
| 暴击伤害 | #ffa500 橙金 | 暴击时替代红色 |
| 护盾吸收 | #aaa 灰 | 被护盾挡掉的部分 |
| 持续伤害 | #ff6600 橙 | DoT 每回合触发 |
| 治疗 | #06d6a0 绿 | 回复 HP |
| 护盾获取 | #fff 白 | 获得护盾量 |
| 反击/反伤 | #ffd93d 黄 | 被动反击伤害 |
| 被动触发 | #7dffb3 青绿 | 被动效果文字 |
| 泡泡 | #4cc9f0 青 | 泡泡盾/泡泡储存 |
| 闪避 | #a0e8ff 浅青 | "闪避!" 文字 |

## 5. 回合系统

### 5.1 行动顺序（批次制）

```
左A → 右CD → 左AB → 右CD → 左AB → ...
      └─回合1─┘  └──回合2──┘  └──回合3──┘
```

- 开局左方先手出 1 只
- 之后双方整队交替行动
- 每两批次(双方各一次) = 1 个回合

### 5.2 回合开始处理

```
1. 技能 CD 全体 -1
2. 被动触发:
   - turnScaleAtk: 攻击成长
   - turnScaleHp: HP成长
   - stoneWall: 防御成长
3. DoT/灼烧 扣血
4. phoenixBurnDot 灼烧(ATK系+%HP系)
5. 熔岩盾 回合递减
6. HOT 持续回复
7. 泡泡储存 回复
8. 泡泡盾到期检查(自然破碎爆炸)
9. 全部 buff turns-1, 到期移除
10. 重算属性(debuff/buff修正)
11. 猎杀被动检查(每批次结束)
```
