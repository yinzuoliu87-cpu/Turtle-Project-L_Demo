# 龟龟对战 — 模块需求文档

## 1. 模块概述

2V2 回合制战斗模块，玩家从已拥有的龟中选择 2 只组队，与 AI 或其他玩家进行战斗。

**入口**: 主页侧栏 → `games/turtle-battle/index.html`

---

## 2. 游戏模式

| 模式 | 说明 | 实现状态 |
|------|------|---------|
| 野生对局 (PvE) | 玩家选 2 只龟，AI 随机选 2 只对战 | ✅ 已完成 |
| 本地双人 | 同一设备轮流选龟、轮流操作 | ✅ 已完成 |
| 线上对战 | 创建/加入 6 位房间号，BroadcastChannel 通信 | ✅ 基础完成 (同浏览器) |

### 线上对战说明
- 当前使用 `BroadcastChannel` 实现同浏览器多标签页对战
- 后续可替换为 WebSocket 服务端实现跨设备对战
- 房间号 6 位随机数字

## 3. 龟龟属性系统

### 3.1 基础属性

| 属性 | 说明 |
|------|------|
| HP | 生命值，归零即阵亡 |
| ATK | 攻击力，影响技能伤害 |
| DEF | 防御力，参与伤害公式分母 |
| SPD | 速度，决定回合行动顺序（高者先手） |
| Crit | 暴击率 (5%~25%)，暴击 = 1.5× 伤害 |
| Res | 抗性，按稀有度自动计算，减少非穿透伤害 |

### 3.2 稀有度体系

每升一级稀有度，基础属性 × 1.03 (即 +3%)，抗性 +1%。

| 稀有度 | 属性倍率 | 抗性 | 颜色 |
|--------|---------|------|------|
| C | ×1.00 | 12% | #06d6a0 绿 |
| B | ×1.03 | 13% | #4cc9f0 蓝 |
| A | ×1.06 | 14% | #3a9abf 深蓝 |
| S | ×1.09 | 15% | #c77dff 紫 |
| SS | ×1.12 | 16% | #ffd93d 金 |
| SSS | ×1.15 | 17% | #ff6b6b 红 |

### 3.3 龟龟总数

共 28 只龟 (C×3, B×8, A×8, S×6, SS×2, SSS×1)，每只有独特技能组合和被动技能。

## 4. 伤害计算公式

```
rawDmg = basePower × (ATK / (ATK + DEF)) × critMult × passiveMult
```

### 4.1 伤害分层

```
原始伤害 (rawDmg)
  ├── 穿透部分 = min(pierce, rawDmg)  ← 无视抗性
  └── 普通部分 = (rawDmg - pierce) × (1 - res%)  ← 受抗性减免

总伤害 = 普通部分 + 穿透部分
  ├── 有护盾 → 先消耗护盾，剩余扣血
  └── 无护盾 → 直接扣血
```

### 4.2 伤害颜色编码

| 类型 | 颜色 | 说明 |
|------|------|------|
| 直接伤害 | #ff4444 红 | 普通伤害扣血部分 |
| 穿透伤害 | #d06bff 紫 | 无视抗性的伤害 |
| 暴击伤害 | #ffa500 橙金 | 暴击时替代红色 |
| 护盾吸收 | #aaa 灰 | 被护盾挡掉的部分 |
| 持续伤害 | #ff6600 橙 | DoT 每回合触发 |
| 治疗 | #06d6a0 绿 | 回复 HP |
| 护盾获取 | #fff 白 | 获得护盾量 |
| 反击伤害 | #ffd93d 黄 | 被动反击 |
| 被动触发 | #7dffb3 青绿 | 被动效果文字 |

## 5. 技能系统

### 5.1 技能类型

| 类型 | 图标 | 目标 | 说明 |
|------|------|------|------|
| physical | ⚔️ | 敌方 | 物理攻击 |
| magic | ✨ | 敌方 | 魔法攻击 |
| heal | 💚 | 己方 | 治疗回复 |
| shield | 🛡 | 己方 | 施加护盾 |

### 5.2 技能属性

| 字段 | 说明 |
|------|------|
| power | 基础威力 |
| hits | 攻击段数 (1~10)，每段独立计算暴击/伤害 |
| pierce | 穿透值，无视抗性和护盾 |
| cd | 冷却回合数 |
| heal | 治疗量 (heal 类型) |
| shield | 护盾量 (shield 类型) |
| random | 是否随机倍率 (×0.5~1.5) |

### 5.3 技能附带效果 (Debuff)

| 效果 | 字段 | 说明 |
|------|------|------|
| 🔥 灼烧 (DoT) | `dot: {dmg, turns}` | 每回合开始造成固定伤害 |
| ⬇ 攻击削减 | `atkDown: {pct, turns}` | 降低攻击力百分比 |
| ⬇ 防御削减 | `defDown: {pct, turns}` | 降低防御力百分比 |
| 🐌 减速 | `spdDown: {pct, turns}` | 降低速度百分比 |

- 同类型不叠加，刷新为更高值/时长
- 回合开始时 tick：DoT 扣血 → 全部 turns-1 → 移除到期 debuff → 重算属性

## 6. 被动技能系统

每只龟有 1 个被动技能，战斗中自动触发。

| 被动类型 | 图标 | 触发时机 | 效果 |
|---------|------|---------|------|
| turnScaleAtk | ⚔️ | 每回合开始 | 基础攻击力 +X% |
| turnScaleHp | 💗 | 每回合开始 | 最大HP +X%（同时回复等量HP） |
| bonusDmgAbove60 | 🎯 | 攻击时 | 对 HP>60% 的目标 +X% 伤害 |
| lowHpCrit | 💢 | 攻击时 | 自身 HP<30% 时暴击率 +X% |
| deathExplode | 💥 | 阵亡时 | 对击杀者造成自身最大 HP ×X% 伤害 |
| shieldOnHit | 🛡 | 被攻击时 | 获得 X 护盾（每回合限 1 次） |
| healOnKill | 💚 | 击杀时 | 恢复自身最大 HP ×X% |
| counterAttack | ⚡ | 被攻击后 | X% 概率反击（造成 50% 基础攻击力伤害） |

---

## 7. 回合流程

```
┌─ 回合开始 ─────────────────────────────────┐
│  1. 技能 CD 全体 -1                         │
│  2. 触发 turnScale 被动 (攻击/HP 成长)       │
│  3. 处理 DoT 伤害 (可能击杀)                 │
│  4. 全部 debuff turns-1, 到期移除            │
│  5. 重算所有属性 (base × debuff 修正)        │
│  6. 按 SPD 排序生成行动队列                  │
│  7. 渲染行动顺序条                           │
│  8. 日志添加回合分隔线                        │
├─ 逐一行动 ─────────────────────────────────┤
│  对每个存活角色:                             │
│  ├─ 玩家控制: 显示技能面板 → 选技能 → 选目标 │
│  └─ AI 控制: 自动选择技能和目标              │
│  执行技能:                                   │
│  ├─ 攻击: 多段伤害循环 (暴击/穿透/抗性/护盾)  │
│  │  ├─ 触发 bonusDmgAbove60 / lowHpCrit     │
│  │  ├─ 触发 shieldOnHit (被攻击方)           │
│  │  ├─ 施加 debuff (dot/atkDown/defDown/spd) │
│  │  └─ 触发 counterAttack (被攻击方)         │
│  ├─ 治疗: 回复目标 HP                        │
│  └─ 护盾: 增加目标护盾值                      │
│  检查死亡:                                    │
│  ├─ 触发 deathExplode (阵亡方)               │
│  └─ 触发 healOnKill (击杀方)                 │
│  检查胜负                                     │
├─ 所有人行动完毕 ───────────────────────────┤
│  turnNum++ → 回到回合开始                    │
└────────────────────────────────────────────┘
```

## 10. AI 策略

| 难度 | 行为 |
|------|------|
| 默认 (normal) | HP<40% 优先治疗 → 无护盾优先加盾 → 随机攻击技能 |

AI 选择目标逻辑:
- 攻击: 选 HP 最低的敌人
- 治疗: 选 HP 百分比最低的队友
- 护盾: 选护盾值最低的队友

## 11. 奖励系统

| 结果 | 奖励 |
|------|------|
| 胜利 | 30 + turnNum×2 龟币 |
| 失败 | 5 龟币 (参与奖) |

- 龟币写入 `localStorage.petState.coins`
- 战绩写入 `localStorage.turtleBattleRecord`
