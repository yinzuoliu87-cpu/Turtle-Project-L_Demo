# 闪电龟 (A级) — 技能设计

## 核心定位
多段魔法输出型。高频攻击叠加电击层数引爆真实伤害，被动每回合自动电击随机敌人，兼具团队增攻能力。

## 被动：雷电

- **每回合自动**电击随机一个敌人，造成 **82%ATK 真实伤害**
- 攻击叠加 **电击层**（每段hit叠1层），满 **8层** 自动引爆，造成 **82%ATK 真实伤害**
- 引爆后层数清零，重新叠加

### 实现要点
- 每回合开始时触发被动电击：随机选敌，造成 0.82×ATK 真实伤害
- 新 debuff 类型 `shockStack`，`{type:'shockStack', stacks:1, maxStacks:8}`
- 满8层时自动引爆，造成 0.82×ATK 真实伤害后清零

---

## 一技能：闪电打击

- **5段伤害**，每段 **23%ATK 魔法伤害**（共115%ATK）
- 每段附带 **25%溅射**（对相邻敌人造成该段25%伤害）
- 每段命中叠加 **1层电击**
- CD: **0**

### 实现要点
```javascript
{ name:'闪电打击', type:'lightningStrike', hits:5, power:0.23, damageType:'magic',
  splash:0.25, desc:'5段23%ATK魔法+25%溅射+叠电击', cd:0 }
```
- 每段 hit 后对目标 increment `shockStack`
- 溅射伤害不叠加电击层

---

## 二技能：威力增幅

- 全体友方 **ATK+15%**，持续 **4回合**
- CD: **4**

### 实现要点
```javascript
{ name:'威力增幅', type:'powerAmplify', target:'allyAll',
  desc:'全体友方ATK+15% 4回合', cd:4 }
```
- 对全体友方 push `{type:'atkUp', value:15, turns:4}`

---

## 三技能：雷暴

- **20段伤害**，每段 **11%ATK 魔法伤害**（共220%ATK），**随机分配**目标
- 每段命中叠加 **1层电击**
- CD: **6**

### 实现要点
```javascript
{ name:'雷暴', type:'thunderstorm', hits:20, power:0.11, damageType:'magic',
  target:'randomEnemy', desc:'20段11%ATK魔法随机+叠电击', cd:6 }
```
- 每段随机选择一个存活敌人，命中后叠加电击层
- 20段可能多次触发满8层引爆

---

## 战斗节奏

1. **每回合**：被动自动电击随机敌人82%ATK真实
2. **常规输出**：闪电打击5段叠电击+溅射
3. **回合2-3**：威力增幅提升全队攻击力15%
4. **爆发回合**：雷暴20段随机轰炸，大量叠层并多次引爆

---

## 平衡考量

- 被动每回合82%ATK真实伤害提供稳定无视防御输出
- 电击引爆（8层82%ATK真实）配合高频攻击可频繁触发
- 雷暴20段可在单回合内多次引爆电击，爆发极高
- 威力增幅为全队增攻15%，辅助价值显著
- SPD18较快，可先手叠层
- 低DEF/MR，需要前排保护
