# 无头龟 (SS级) — 技能设计

## 核心定位
亡灵战士型。自带吸血、低血高攻、濒死锁血机制，越残血越危险。技能组包含百分比伤害、恐惧控制和全体收割。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 350 |
| ATK | 39 |
| DEF | 13 |
| MR | 12 |
| SPD | 16 |
| 暴击 | 25% |

---

## 被动：亡灵

- **登场时**获得 **+22%吸血**（所有造成的伤害均生效）
- **血越低ATK越高**：每损失1%HP -> ATK+1%，最高+100%ATK（即1HP时ATK翻倍）
- **首次濒死锁血**：第一次HP降到0时，锁定1HP，持续 **2回合**（期间不会死亡）

### 实现要点
- 被动类型 `undead`
- 吸血：在 `triggerOnHitEffects` 中检测被动，回复伤害的22%
- 低血增攻：在计算ATK时动态加成 `atkBonus = Math.floor((1 - hp/maxHp) * 100)`%，上限100%
- 濒死锁血：在 `checkDeaths` 中检测，首次HP<=0时设置 `hp=1`，标记 `_undeadLockTurns=2`
- 2回合后锁血解除，再次HP<=0则正常阵亡

## 一技能：撕咬

- **2段**，每段 **65%×ATK + 4%目标HP** 物理伤害
- 共 **130%×ATK + 8%目标HP**
- CD: 0

### 实现要点
```javascript
{ name:'撕咬', type:'physical', hits:2, power:0, pierce:0,
  desc:'2段65%ATK+4%目标HP物理伤害',
  cd:0, atkScale:0.65, hpPct:4 }
```
- 每段独立暴击判定
- 每段触发22%吸血回复

---

## 二技能：恐吓

- 造成 **90%×ATK** 物理伤害
- 施加 **恐惧** 3回合：被恐惧目标伤害 **-20%**
- CD: 3

### 实现要点
```javascript
{ name:'恐吓', type:'headlessFear', hits:1, power:0, pierce:0,
  desc:'90%ATK物理伤害+恐惧3回合-20%伤害',
  cd:3, atkScale:0.9, fearTurns:3, fearDmgDown:20 }
```
- 技能类型 `headlessFear`
- 先造成物理伤害，然后对目标施加恐惧 debuff
- 恐惧效果：被恐惧单位所有造成的伤害-20%，持续3回合

---

## 三技能：灵魂收割

- 对**全体敌方**造成 **110%×ATK + 10%目标已损HP** 物理伤害
- CD: 4

### 实现要点
```javascript
{ name:'灵魂收割', type:'headlessReap', hits:1, power:0, pierce:0,
  desc:'全体110%ATK+10%已损HP物理伤害',
  cd:4, atkScale:1.1, lostHpPct:10, aoe:true }
```
- 技能类型 `headlessReap`
- 对每个敌人分别计算已损HP比例伤害
- 配合低血高攻被动，残血时此技能伤害极高（ATK翻倍 + 目标已损HP）

## 平衡考量

- **22%吸血**：配合持续输出可以回大量血，续航能力强
- **低血高攻**：被动让无头龟越残血越危险，敌方需要一波秒杀而非慢慢磨
- **濒死锁血2回合**：给无头龟一个翻盘窗口，锁血期间ATK接近+100%，配合吸血可能拉回血量
- **恐惧-20%伤害**：降低敌方输出，间接提高自身生存
- **灵魂收割**：全体+已损HP比例，作为收割技能越晚放越强
- **弱点**：DEF13/MR12偏低，被集火容易触发锁血；锁血后2回合若无法回血则必死
