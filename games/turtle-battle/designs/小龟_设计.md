# 小龟 (C级) — 技能设计

## 核心定位
万金油输出型。被动对高稀有度目标有天然增伤，越高级的敌人越疼。技能组兼具自增益、护盾和多段AOE输出。

---

## 被动：基本龟

- 根据敌人稀有度造成额外伤害加成：
  - C级：+0%
  - B级：+8%
  - A级：+16%
  - S级：+24%
  - SS级：+32%
  - SSS级：+48%
- 适用于所有造成伤害的技能

### 实现要点
- 新被动类型 `basicTurtle`
- 增伤映射表：`{ C:0, B:8, A:16, S:24, SS:32, SSS:48 }`
- 在 `doDamage` 中检测被动类型，根据目标 rarity 乘以对应增伤
- 日志中显示增伤百分比

---

## 一技能：攻击

- **2段**共 **1.2×ATK** 普通伤害
- 自身获得 **ATK+15%**，持续 **2回合**
- CD: 0

### 实现要点
```javascript
{ name:'攻击', type:'physical', hits:2, power:0, pierce:0,
  desc:'2段共1.2×ATK普通伤害，自身ATK+15% 2回合',
  cd:0, atkScale:1.2, selfAtkUpPct:{pct:15, turns:2} }
```
- 复用 physical 类型 + 新参数 `selfAtkUpPct`
- 在 `doDamage` 末尾（类似 `selfDefUpPct` 的处理方式）给自己加 atkUp buff

---

## 二技能：龟盾

- 撞击敌人造成 **0.8×ATK** 普通伤害
- 获得 **造成伤害50%** 的 **永久护盾**（不会过期）
- CD: 2

### 实现要点
```javascript
{ name:'龟盾', type:'turtleShieldBash', hits:1, power:0, pierce:0,
  desc:'0.8×ATK伤害，获得50%伤害值的永久护盾',
  cd:2, atkScale:0.8, shieldFromDmgPct:50 }
```
- 新技能类型 `turtleShieldBash`
- 自定义函数：先计算伤害，然后 `shieldGain = Math.round(actualDmg * 0.5)`
- 护盾不设 turns，永久存在

---

## 三技能：打击

- **10段伤害**随机分布到所有敌人
- 共 **2.5×ATK** 普通伤害
- CD: 5

### 实现要点
```javascript
{ name:'打击', type:'basicBarrage', hits:10, power:0, pierce:0,
  desc:'10段共2.5×ATK随机分布敌方',
  cd:5, atkScale:2.5 }
```
- 新技能类型 `basicBarrage`（类似猎人箭雨/闪电雷暴）
- 每段 0.25×ATK，随机选择一个存活敌人命中
- 每段独立暴击判定，触发被动增伤

## 伤害模拟（小龟 ATK=38，对S级目标DEF=15）

### 被动增伤对比
| 目标稀有度 | 增伤 | 一技能理论伤害 |
|-----------|------|---------------|
| C级 | +0% | ~30 |
| B级 | +8% | ~32 |
| A级 | +16% | ~35 |
| S级 | +24% | ~37 |
| SS级 | +32% | ~40 |
| SSS级 | +48% | ~44 |

### 三技能：打击（10段分布）
- 每段 0.25×38≈10 → 减防后≈7
- 对S级+24% → 每段≈9
- 10段总伤≈90（分散到2个敌人≈每人45）

## 平衡考量

- **被动增伤按稀有度递增**：C级小龟打C级无加成，但打SS/SSS有可观增伤，体现"以弱胜强"
- **一技能无CD+自buff**：持续输出+自增益，越打越强
- **二技能护盾转化**：打得越疼盾越厚，鼓励进攻
- **三技能10段随机**：高总伤但分散，对双目标场景效率高
- **弱点**：基础属性低（C级），面对同级无增伤加持
