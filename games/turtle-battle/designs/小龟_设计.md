# 小龟 (C级) — 技能设计

## 核心定位
万金油输出型。被动对高稀有度目标有天然增伤，越高级的敌人越疼。技能组兼具自增益、护盾和多段AOE输出。

## 被动：不屈

- 根据目标稀有度造成额外伤害加成：
  - C级：+20%
  - B级：+23%
  - A级：+26%
  - S级：+29%
  - SS级：+32%
  - SSS级：+34%
- 适用于所有造成伤害的技能

### 实现要点
- 被动类型 `basicTurtle`
- 增伤映射表：`{ C:20, B:23, A:26, S:29, SS:32, SSS:34 }`
- 在 `doDamage` 中检测被动类型，根据目标 rarity 乘以对应增伤
- 日志中显示增伤百分比

## 一技能：攻击

- **2段**共 **70%×ATK×2 = 140%ATK** 物理伤害
- 自身获得 **ATK+20%**，持续 **2回合**
- CD: 0

### 实现要点
```javascript
{ name:'攻击', type:'physical', hits:2, power:0, pierce:0,
  desc:'2段70%ATK物理伤害，自身ATK+20% 2回合',
  cd:0, atkScale:0.7, selfAtkUpPct:{pct:20, turns:2} }
```
- 复用 physical 类型 + `selfAtkUpPct`
- 在 `doDamage` 末尾给自己加 atkUp buff

---

## 二技能：龟盾

- 撞击敌人造成 **70%×ATK + 目标已损HP×20%** 物理伤害
- 获得 **造成伤害80%** 的 **永久护盾**（不会过期）
- CD: 2

### 实现要点
```javascript
{ name:'龟盾', type:'turtleShieldBash', hits:1, power:0, pierce:0,
  desc:'70%ATK+目标已损HP20%物理伤害，获得80%伤害值永久护盾',
  cd:2, atkScale:0.7, lostHpPct:20, shieldFromDmgPct:80 }
```
- 技能类型 `turtleShieldBash`
- 先计算伤害（70%ATK + 目标已损HP×20%），然后 `shieldGain = Math.round(actualDmg * 0.8)`
- 护盾不设 turns，永久存在

---

## 三技能：打击

- **10段伤害**随机分布到所有敌人
- 共 **290%×ATK** 物理伤害
- CD: 4

### 实现要点
```javascript
{ name:'打击', type:'basicBarrage', hits:10, power:0, pierce:0,
  desc:'10段共290%ATK物理随机分布敌方',
  cd:4, atkScale:2.9 }
```
- 技能类型 `basicBarrage`（类似猎人箭雨/闪电雷暴）
- 每段 29%×ATK，随机选择一个存活敌人命中
- 每段独立暴击判定，触发被动增伤

## 平衡考量

- **被动不屈按稀有度递增**：对所有等级目标均有增伤（C+20%起步），对SS/SSS有更高增伤，体现"以弱胜强"
- **一技能无CD+自buff**：持续输出+自增益ATK+20%，越打越强
- **二技能护盾转化**：包含目标已损HP比例伤害，打残血目标更疼；80%伤害转护盾，鼓励进攻
- **三技能10段随机**：290%ATK高总伤但分散，对多目标场景效率高，CD4较短
- **弱点**：基础属性低（C级），但被动保底+20%增伤弥补了部分差距
