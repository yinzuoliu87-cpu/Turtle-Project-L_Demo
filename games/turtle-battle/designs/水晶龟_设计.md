# 水晶龟 (S级) — 技能设计

## 核心定位
坦克/破抗型。高双抗+魔法减免，通过结晶印记引爆%HP伤害并削弱敌方魔抗，兼具全队防御辅助。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 382 |
| ATK | 44 |
| DEF | 21 |
| MR | 23 |
| SPD | 12 |
| 暴击 | 25% |

---

## 被动：水晶共鸣

- 受到的 **魔法伤害额外减免20%**（在魔抗减免之后再乘0.8）
- 攻击叠加 **结晶印记**，满 **4层** 自动引爆：
  - 造成 **10%目标最大HP** 的 **魔法伤害**
  - 降低目标 **MR-20%**，持续 **3回合**

### 实现要点
- 在 `doDamage` 中，若伤害类型为 magic 且目标为水晶龟，最终伤害 ×0.8
- 新 debuff 类型 `crystalMark`，`{type:'crystalMark', stacks:1, maxStacks:4}`
- 满4层引爆：造成 target.maxHP×0.10 魔法伤害，push `{type:'mrDown', value:20, turns:3}`，清零层数

---

## 一技能：水晶刺

- **2段伤害**，每段 **50%ATK + 3%目标当前HP** 的 **魔法伤害**（共100%ATK+6%HP）
- 每段叠加 **1层结晶印记**
- CD: **0**

### 实现要点
```javascript
{ name:'水晶刺', type:'crystalSpike', hits:2, power:0.50, bonusHpScale:0.03,
  damageType:'magic', desc:'2段50%ATK+3%目标HP魔法+叠结晶', cd:0 }
```
- 每段 hit 后 increment `crystalMark`

---

## 二技能：水晶壁垒

- 获得 **90%ATK 护盾**
- 全体友方 **DEF+15%** 和 **MR+15%**，持续 **3回合**
- CD: **3**

### 实现要点
```javascript
{ name:'水晶壁垒', type:'crystalBarrier', shieldScale:0.90,
  desc:'90%ATK护盾+全体+15%DEF/MR 3回合', cd:3 }
```
- 自身获得护盾
- 全体友方 push `{type:'defUp', value:15, turns:3}`, `{type:'mrUp', value:15, turns:3}`

---

## 三技能：碎晶爆破

- 对全体敌人造成 **5段**，每段 **16%ATK 魔法伤害** + **8%ATK 真实伤害**（共80%ATK魔法+40%ATK真实）
- 每段叠加 **1层结晶印记**
- CD: **4**

### 实现要点
```javascript
{ name:'碎晶爆破', type:'crystalBlast', hits:5, power:0.16, truePower:0.08,
  damageType:'magic', target:'allEnemy',
  desc:'全体5段16%ATK魔+8%ATK真+叠结晶', cd:4 }
```
- 每段对全体敌人造成魔法+真实伤害
- 每段命中叠加结晶印记（5段可能触发1-2次引爆）

---

## 战斗节奏

1. **常规回合**：水晶刺持续输出+叠印记（2回合可引爆一次）
2. **防御回合**：水晶壁垒给全队加防+自身护盾
3. **爆发回合**：碎晶爆破全体5段+大量叠层，可多次引爆
4. 引爆提供持续的%HP伤害和MR削弱，越打越疼

---

## 平衡考量

- HP382+DEF21+MR23为最高坦度之一，加上被动魔法减免20%，极难被魔法击杀
- 结晶引爆10%maxHP对高HP目标特别有效
- 水晶壁垒为全队提供双抗提升，辅助坦度极高
- 碎晶爆破5段全体可大量叠层并触发引爆
- SPD12偏慢，输出节奏较缓
- ATK44中等，纯ATK倍率伤害不突出，依赖%HP和引爆机制
