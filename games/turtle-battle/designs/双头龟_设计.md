# 双头龟 (B级) — 技能设计

## 核心定位
双形态切换型。远程高攻低防 <-> 近战低攻高防高血，切换时属性实时变化。

---

## 属性（远程形态 = 默认）
| 属性 | 远程 | 近战（变化） |
|------|------|------------|
| HP | 302 | +150%×ATK 最大HP（当前HP按比例缩放） |
| ATK | 50 | -30%ATK |
| DEF | 11 | +25%×ATK 防御 |
| MR | 12 | 不变 |
| SPD | 7 | 不变 |
| 暴击 | 25% | 不变 |

### 形态切换属性计算

**远程 -> 近战**（以ATK=50为例）：
1. `hpGain = ATK × 1.5 = 75`，`maxHp += hpGain`（302->377），当前HP按比例缩放
2. `defGain = ATK × 0.25 = 12`，`baseDef += defGain`（11->23）
3. `atkLoss = ATK × 0.30 = 15`，`baseAtk -= atkLoss`（50->35）
4. 获得 **110%×ATK 永久护盾**（55护盾）

**近战 -> 远程**：
1. `maxHp -= hpGain`，当前HP按比例缩放（不超过新maxHp）
2. `baseDef -= defGain`
3. `baseAtk += atkLoss`

## 远程形态技能

### 一技能：魔法波
- **4段** 物理伤害和真实伤害交替
- 每段 40%×ATK
- 共 160%×ATK
- 奇数段 = 物理伤害，偶数段 = 真实伤害
- CD: 0

### 二技能：灵能冲击
- 对**全体敌方**造成 **85%×ATK + 15%目标HP** 物理伤害
- CD: 4

### 三技能：切换近战
- 变为近战形态
- 属性变化：+150%ATK HP、+25%ATK DEF、-30%ATK
- 获得 **110%×ATK 永久护盾**
- 对目标造成 **120%×ATK** 物理伤害
- CD: 4

## 实现要点

### 形态状态
```javascript
f._twoHeadForm = 'ranged';  // 'ranged' | 'melee'
f._formHpGain = 0;          // 记录的HP变化值
f._formDefGain = 0;         // 记录的DEF变化值
f._formAtkLoss = 0;         // 记录的ATK变化值
f._rangedSkills = [...];    // 保存远程技能组
f._meleeSkills = [...];     // 保存近战技能组
```

### 切换逻辑 (doTwoHeadSwitch)
```
if switchTo === 'melee':
  1. 记录变化值：hpGain=ATK×1.5, defGain=ATK×0.25, atkLoss=ATK×0.30
  2. maxHp += hpGain, hp按比例缩放
  3. baseDef += defGain
  4. baseAtk -= atkLoss
  5. f.skills = f._meleeSkills
  6. 获得110%×ATK永久护盾
  7. 造成120%×ATK物理伤害
  
if switchTo === 'ranged':
  1. 恢复记录的变化值
  2. maxHp, hp, baseDef, baseAtk还原
  3. f.skills = f._rangedSkills
  4. 打出1.4×ATK攻击 + 减防25%ATK 4回合
```

### 魔法波 (twoHeadMagicWave)
- 4段循环：段1物理、段2真实、段3物理、段4真实
- 物理段走DEF减免，真实段无视DEF和MR

### 吸收 (twoHeadAbsorb)
- 伤害 = 60%×ATK + 8%目标HP
- 回复 = 40%×ATK + (maxHp - hp) × 18%

## 平衡考量

- 远程：ATK50高攻但HP302/DEF11低防，纯输出型
- 近战：ATK35(-30%)但DEF23(+12)、HP377(+75)、+55永久护盾，较肉
- 切换有4回合CD，不能频繁切
- 近战->远程切换时打出一发攻击+减防，创造输出窗口
- 远程->近战切换时获得110%ATK永久护盾+120%ATK攻击，攻防转换有爆发
- 近战锤击自带50%伤害转护盾，持续作战能力强
