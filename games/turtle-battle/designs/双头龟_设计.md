# 双头龟 (B级) — 技能设计

## 核心定位
双形态切换型。远程高攻低防 ↔ 近战低攻高防高血，切换时属性实时变化。

---

## 基础属性（远程形态 = 默认）

| 属性 | 远程 | 近战（变化） |
|------|------|------------|
| HP | 280 | +150%×ATK 最大HP（当前HP按比例缩放） |
| ATK | 49 | -30%ATK |
| DEF | 10 | +15%×ATK 防御 |

### 形态切换属性计算

**远程 → 近战**（以ATK=49为例）：
1. 记录变化值：`hpGain = ATK × 1.5 = 74`，`defGain = ATK × 15% = 7`，`atkLoss = ATK × 30% = 15`
2. `maxHp += hpGain`（280→354），当前HP按比例缩放
3. `baseDef += defGain`（10→17）
4. `baseAtk -= atkLoss`（49→34）
5. 获得 100%×ATK 护盾（49护盾）

**近战 → 远程**：
1. `maxHp -= hpGain`，当前HP按比例缩放（不超过新maxHp）
2. `baseDef -= defGain`
3. `baseAtk += atkLoss`

---

## 被动：双头

- 拥有两套技能组（远程/近战），通过三技能切换
- 默认远程形态
- 切换时属性实时变化

---

## 远程形态技能

### 一技能：魔法波
- **4段** 普通伤害和穿透伤害交替
- 共 160%×ATK
- 每段 40%×ATK，奇数段 = 普通伤害，偶数段 = 穿透伤害
- CD: 0

### 二技能：灵能冲击
- 对**全体敌方**造成 80%×ATK + 9%目标最大HP 普通伤害
- CD: 4

### 三技能：切换近战
- 变为近战形态
- 属性变化：+100%HP、+10%ATK转防御、-20%ATK
- 变形后获得 **15%最大HP的护盾**（持续4回合）
- CD: 2

---

## 近战形态技能

### 一技能：锤击
- 造成 **140%×ATK** 普通伤害
- CD: 0

### 二技能：吸收
- 造成 **50%×ATK + 5%目标最大HP** 普通伤害
- 回复自身 **12%已损生命值**
- CD: 2

### 三技能：切换远程
- 变为远程形态
- 属性还原
- 变形时打出一发远程攻击：**140%×ATK** 普通伤害
- 并减少目标 **10%双头龟ATK** 的防御值，持续4回合
- CD: 2

## 实现要点

### 形态状态
```javascript
f._twoHeadForm = 'ranged';  // 'ranged' | 'melee'
f._formHpGain = 0;          // 记录的HP变化值
f._formDefGain = 0;         // 记录的DEF变化值
f._formAtkLoss = 0;         // 记录的ATK变化值
f._rangedSkills = [...];    // 保存远程技能组
f._meleeSkills = [...];     // 保存近战技能组
```

### 切换逻辑 (doTwoHeadSwitch)
```
if switchTo === 'melee':
  1. 记录变化值：hpGain=ATK×1.5, defGain=ATK×15%, atkLoss=ATK×30%
  2. maxHp += hpGain, hp按比例缩放
  3. baseDef += defGain
  4. baseAtk -= atkLoss
  5. f.skills = f._meleeSkills
  6. 获得100%×ATK护盾
  
if switchTo === 'ranged':
  1. 恢复记录的变化值
  2. maxHp, hp, baseDef, baseAtk还原
  3. f.skills = f._rangedSkills
  4. 打出1.4×ATK攻击 + 减防
```

### 魔法波 (twoHeadMagicWave)
- 4段循环：段1普通、段2穿透、段3普通、段4穿透
- 普通段走DEF减免，穿透段无视DEF

### 吸收 (twoHeadAbsorb)
- 伤害 = 0.5×ATK + 5%目标HP
- 回复 = (maxHp - hp) × 12%

---

## 平衡考量

- 远程：ATK49高攻但HP280/DEF10低防，纯输出型
- 近战：ATK34(-30%)但DEF17(+7)、HP354(+74)、+49护盾，较肉
- 切换有3回合CD，不能频繁切
- 近战→远程切换时打出一发攻击+减防，创造输出窗口
- 远程→近战切换时获得ATK值的护盾，缓冲形态转换
