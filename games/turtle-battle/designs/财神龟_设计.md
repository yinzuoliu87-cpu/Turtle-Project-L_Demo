# 财神龟 (B级) — 技能设计

## 核心定位
金币赌博型。通过骰子积攒金币和回复，最终梭哈消耗全部金币造成全体毁灭性伤害，一场仅一次的终极赌注。

## 被动：聚宝盆

- **每回合**自动获得 **3~8** 金币（随机）
- 任意单位 **阵亡**（敌我均计）时获得额外 **+9金币**

### 实现要点
- 新资源 `gold`，无上限，初始0
- 每回合开始时 gold += random(3, 8)
- 在 `onDeath` 事件中，无论敌我阵亡，财神龟 gold += 9
- UI显示当前金币数

---

## 一技能：打击

- **3段伤害**，每段 **45%ATK 物理伤害**（共135%ATK）
- CD: **0**

### 实现要点
```javascript
{ name:'打击', type:'strike', hits:3, power:0.45, damageType:'physical',
  desc:'3段45%ATK物理', cd:0 }
```

---

## 二技能：骰子

- 获得 **3~8金币**（随机）
- 回复 **13%最大HP + 15%已损失HP**
- 若 **梭哈已使用**，额外获得 **10%HP 护盾**
- CD: **0**

### 实现要点
```javascript
{ name:'骰子', type:'dice',
  desc:'+3~8金币+13%HP回复+15%已损HP,梭哈后额外10%HP护盾', cd:0 }
```
- gold += random(3, 8)
- 回复 maxHP×0.13 + (maxHP - currentHP)×0.15
- 检查 `allInUsed` 标记，若为 true 则额外获得 maxHP×0.10 护盾

---

## 三技能：梭哈

- **消耗全部金币**
- 每枚金币造成 **30%ATK 物理伤害 + 30%ATK 真实伤害** 对 **全体敌人**
- **一场战斗限用一次**（CD: 999）
- 例：50金币时，全体每人受到 50×(30%+30%)ATK = 30×ATK 伤害

### 实现要点
```javascript
{ name:'梭哈', type:'allIn', target:'allEnemy',
  desc:'消耗全部金币,每枚30%ATK物理+30%ATK真实全体,一场限一次', cd:999 }
```
- 伤害 = gold × (ATK×0.30 物理 + ATK×0.30 真实)，对全体敌人
- 使用后 gold = 0，设置 `allInUsed = true`
- CD设为999确保无法再次使用

---

## 战斗节奏

1. **前期**：被动+骰子积攒金币（每回合约6~16金币）
2. **过渡期**：打击维持输出，骰子回复续航
3. **关键时刻**：梭哈释放全部金币，全体毁灭性伤害
4. **梭哈后**：骰子获得额外护盾，继续打击+骰子维持
5. 阵亡事件触发额外金币，战斗越激烈积累越快

---

## 平衡考量

- 梭哈为一场一次的终极爆发，时机选择至关重要
- 假设积攒40金币梭哈：全体每人受 40×0.60×39 = 936伤害（物理+真实各一半）
- 骰子提供稳定回复和金币，使其不完全依赖打击输出
- 被动每回合3~8金币+阵亡9金币，积累速度中等
- 梭哈过早金币少伤害低，过晚可能己方已劣势
- SPD10中等偏慢，给对手留有反应时间
- 梭哈后失去核心爆发手段，靠打击和骰子维持
