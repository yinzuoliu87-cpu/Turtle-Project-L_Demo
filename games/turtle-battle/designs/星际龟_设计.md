# 星际龟 (S级) — 技能设计

## 核心定位
远程爆发 + 能量蓄力型。通过蓄能机制在关键时刻释放超高伤害，带有空间控制和持续打击能力。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 320 |
| ATK | 44 |
| DEF | 12 |
| 暴击 | 25% |

---

## 被动：星能充能

- 星际龟每次造成伤害时积累**星能**（造成伤害的 20%）
- 星能上限 = 最大HP的 50%（即 160）
- 当星能满时，下一次攻击技能自动附加**星能爆发**：消耗全部星能，额外造成等量的**穿透伤害**
- 卡片上显示星能条（类似泡泡龟的泡泡储存条，用金色）

### 设计意图
- 鼓励持续输出积累能量
- 满能后的一次攻击会有巨额穿透爆发（160穿透 ≈ 3.6×ATK）
- 对方可以通过护盾来挡穿透，但不能用防御减免

---

## 一技能：星光射线

- 对单体目标造成 **3段** 伤害
- 每段 **40%×ATK** 普通伤害 + **5%目标当前HP** 普通伤害
- 共 120%×ATK + 15%目标当前HP
- CD: 0

### 设计意图
- 基于目标当前HP的伤害：对高血目标伤害更高，越打越低
- 配合被动：3段都积累星能

---

## 二技能：虫洞

- 对敌方单体施加**虫洞标记** 3回合
- 被标记的目标受到的所有穿透伤害 **+30%**
- 标记期间，星际龟对该目标的普通伤害 **+15%**
- CD: 3

### 设计意图
- 穿透伤害增幅配合被动的星能爆发（穿透×1.3 = 巨额伤害）
- 也增幅队友的穿透伤害（如猎人龟连珠箭）
- 是一个策略性debuff，标记后再释放星能爆发效果最大

---

## 三技能：流星暴击

- 对**全体敌方**造成 **60%×ATK** 普通伤害
- 并消耗当前 **50%星能** 转为额外穿透伤害（平均分配到每个目标）
- 施加 15% 防御削减 2回合
- CD: 4

### 设计意图
- AOE + 星能消耗 = 不如攒满被动爆发伤害高，但能群伤
- 防御削减为队友后续输出创造机会
- 是一个"不够满能时的清场选择"

## 战斗节奏

```
回合1: 星光射线（积累星能 ~26）
回合2: 虫洞标记敌人（不积累）
回合3: 星光射线（积累 ~26，总 ~52）
回合4: 星光射线（积累 ~26，总 ~78）
回合5: 星光射线（积累 ~26，总 ~104）
...
回合N: 星能满（160），下次攻击自动爆发 → 160穿透 × 虫洞1.3 = 208穿透！
```

或者：
```
积累到 80 星能时用流星暴击 → 消耗 40 星能 → 全体60%ATK + 40穿透 + 减防
```

---

## 平衡考量

- 星能积累速度取决于输出量，被护盾/防御减免的不算（只算实际造成伤害）
- 星能爆发是穿透伤害 → 打护盾但无视防御
- 虫洞标记让星能爆发更恐怖 → 但虫洞本身不造成伤害（占一回合）
- HP320 + DEF12 比较脆 → 需要队友保护才能蓄力到满
- 如果星际龟被集火秒杀 → 星能白费
- 流星暴击消耗50%星能 → 是一个"提前变现"的选择
