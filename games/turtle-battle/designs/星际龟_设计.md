# 星际龟 (S级) — 技能设计

## 核心定位
远程爆发 + 能量蓄力型。通过星能机制在关键时刻释放超高伤害，带有空间控制和持续打击能力。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 349 |
| ATK | 45 |
| DEF | 13 |
| MR | 15 |
| SPD | 17 |
| 暴击 | 25% |

---

## 被动：星能

- 星际龟每次造成伤害时积累**星能**（造成伤害的 **62%**）
- 星能上限 = 最大HP的 **40%**（即 139.6）
- 每次技能后，消耗 **30%星能** 造成等量的**真实伤害**
- 三技能"流星暴击"满能时消耗 **100%星能** 附加真实伤害
- 卡片上显示星能条（金色）

### 设计意图
- 鼓励持续输出积累能量
- 每次技能后都有真实伤害输出（30%星能），保持持续压力
- 满能流星暴击可一波释放全部星能

---

## 一技能：星光射线

- 对单体目标造成 **3段** 伤害
- 每段 **40%×ATK** 魔法伤害 + **6%目标当前HP** 魔法伤害
- 共 120%×ATK + 18%目标当前HP
- CD: 0

### 设计意图
- 基于目标当前HP的伤害：对高血目标伤害更高，越打越低
- 配合被动：3段都积累星能

---

## 二技能：虫洞

- 对敌方单体施加**虫洞标记** **4回合**
- 被标记的目标受到的真实伤害 **+20%**
- 并永久获得 **10%×ATK** 的魔穿
- CD: 4

### 设计意图
- 真实伤害增幅配合被动的星能释放（真实×1.2 = 大量伤害）
- 永久魔穿提升后续所有魔法伤害
- 是一个策略性debuff，标记后再释放星能效果最大

---

## 三技能：流星暴击

- 对**全体敌方**造成 **100%×ATK** 魔法伤害
- 如果星能满，消耗 **100%星能** 转为额外真实伤害
- 施加 **-20% MR**，持续 **3回合**
- CD: 4

### 设计意图
- AOE + 满能附加真实 = 巨额爆发
- MR削减为后续魔法伤害创造机会
- 满能释放时机很关键

## 战斗节奏

```
回合1: 星光射线（积累星能，技能后释放30%星能真实伤害）
回合2: 虫洞标记敌人（提高真实伤害+20%，获得魔穿）
回合3: 星光射线（继续积累，每次技能后释放30%）
回合4: 星光射线（继续积累）
...
回合N: 星能满 -> 流星暴击 -> 消耗100%星能，全体魔法+真实爆发！
```

---

## 平衡考量

- 星能积累速度 = 造伤的62%，积累较快
- 上限40%HP = 约140星能，满能后流星暴击附加140真实伤害（分摊到全体）
- 每次技能后30%星能释放 = 持续输出压力，不需要等满能
- 虫洞标记+20%真实伤害 -> 配合被动更恐怖
- HP349 + DEF13 比较脆 -> 需要队友保护才能持续输出
- SPD17极高 -> 通常先手行动

## 依赖

- `_starEnergy` 字段（星能值）
- 星能条 UI（金色条）
- `starWormhole` debuff 类型（增幅真实伤害+永久魔穿）
- 每次技能后自动释放30%星能真实伤害
- 流星暴击满能消耗逻辑
