# 寒冰龟 (B级) — 技能设计

## 核心定位
控制/减益型。登场即削弱全体敌方攻击力，免疫灼烧，对火系龟有额外克制，兼具护盾辅助和AOE魔穿能力。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 381 |
| ATK | 41 |
| DEF | 16 |
| MR | 18 |
| SPD | 9 |
| 暴击 | 25% |

---

## 被动：冰寒

- **登场时**对敌方所有龟施加 **ATK-20%**，持续 **6回合**
- **免疫灼烧**状态
- 对 **火系龟** 造成 **+35%伤害**（所有技能生效）
- 被动减攻在战斗开始时自动触发，不占回合

### 实现要点
- 被动类型 `frostAura`
- 在 `startBattle` 中，遍历 allFighters，对敌方全体 push `atkDown` debuff `{type:'atkDown', value:20, turns:6}`
- 免疫灼烧：在施加灼烧时检测被动，跳过
- 伤害增幅：在 `doDamage` 中检测攻击者被动 `frostAura`，若目标为火系龟，则最终伤害 x1.35
- 日志：`寒冰龟 被动：敌方全体ATK-20% 6回合`

## 一技能：冰锥

- **6段伤害**，共 **140%×ATK**
- 物理伤害和魔法伤害**交替**：
  - 第1段：物理（约23%×ATK，受DEF减免）
  - 第2段：魔法（约23%×ATK，受MR减免）
  - 第3段：物理
  - 第4段：魔法
  - 第5段：物理
  - 第6段：魔法
- CD: 0

### 实现要点
```javascript
{ name:'冰锥', type:'iceSpike', hits:6, power:0, pierce:0,
  desc:'6段物理/魔法交替共140%ATK', cd:0, totalScale:1.4 }
```
- 技能类型 `iceSpike`
- 6段循环，每段约23.3%×ATK
  - 奇数段（1,3,5）：物理伤害，走 DEF 减免
  - 偶数段（2,4,6）：魔法伤害，走 MR 减免
- 每段独立暴击判定
- 每段触发 on-hit 被动

---

## 二技能：冰盾

- 为**自身**获得 **140%×ATK** 永久护盾
- 为**友方目标**获得 **80%×ATK** 永久护盾
- CD: 2

### 实现要点
```javascript
{ name:'冰盾', type:'iceShield', hits:1, power:0, pierce:0,
  desc:'自身140%ATK+友方80%ATK永久护盾', cd:2,
  selfShieldScale:1.4, allyShieldScale:0.8 }
```
- 技能类型 `iceShield`
- 自身护盾 = `Math.round(caster.atk * 1.4)`
- 友方护盾 = `Math.round(caster.atk * 0.8)`
- 护盾为永久（不设 turns）

---

## 三技能：冰霜

- 对**所有敌人**施加 **MR-25%**，持续 **4回合**
- 随后对全体敌人造成 **10段** 共 **180%×ATK（10×18%ATK）** 魔法伤害
- CD: 4

### 实现要点
```javascript
{ name:'冰霜', type:'iceFrost', hits:10, power:0, pierce:0,
  desc:'全体-25%MR 4回合 + 10段18%ATK魔法伤害',
  cd:4, mrDownPct:25, mrDownTurns:4, atkScale:0.18, aoe:true }
```
- 技能类型 `iceFrost`
- 先对全体敌方施加 MR-25% debuff 4回合
- 然后对全体造成10段魔法伤害（每段18%ATK），受MR减免（已被削减后的MR）

## 平衡考量

- **被动ATK-20%持续6回合**：开场控制强，削弱幅度大，但时间有限
- **免疫灼烧**：对火系龟的额外防御机制
- **对火系+35%增伤**：定向克制火系龟，寒冰龟成为火系天敌
- **一技能6段物理/魔法交替**：多段触发各种 on-hit 效果，交替伤害类型让对方难以用单一防御应对
- **二技能冰盾**：自身140%ATK + 友方80%ATK永久护盾，防御辅助兼备
- **三技能AOE魔穿**：先削MR再打魔法伤害（每段18%ATK），自身联动效果好，10段多段触发

## 与其他龟的协同/克制

### 克制
- **火系龟（熔岩龟/凤凰龟）**：被动+35%增伤+免疫灼烧，天然克制
- **高攻龟**：被动-20%ATK大幅削弱对方输出

### 被克制
- **闪电龟**：多段电击对寒冰龟消耗快
- **天使龟**：审判被动按当前HP计算，寒冰龟HP较高容易被吃

### 协同
- 配合石头龟：冰盾+石头龟岩石护甲，双重护盾非常肉
- 配合天使龟：6段冰锥每段触发审判被动，爆发惊人
