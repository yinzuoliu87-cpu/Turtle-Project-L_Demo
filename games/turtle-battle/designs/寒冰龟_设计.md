# 寒冰龟 (B级) — 技能设计

## 核心定位
控制/减益型。登场即削弱全体敌方攻击力，对火系龟有额外克制，兼具护盾辅助和AOE穿透能力。

---

## 被动：冰寒

- **登场时**对敌方所有龟施加 **ATK-15%**，持续 **6回合**
- 对 **熔岩龟** 和 **凤凰龟** 造成 **+20%伤害**（所有技能生效）
- 被动减攻在战斗开始时自动触发，不占回合

### 实现要点
- 新被动类型 `frostAura`
- 在 `startBattle` 中，遍历 allFighters，对敌方全体 push `atkDown` debuff `{type:'atkDown', value:15, turns:6}`
- 伤害增幅：在 `doDamage` 中检测攻击者被动 `frostAura`，若目标 id 为 `lava` 或 `phoenix`，则最终伤害 ×1.2
- 日志：`❄️寒冰龟 被动：敌方全体ATK-15% 6回合`

---

## 一技能：冰锥

- **6段伤害**，共 **1.2×ATK**
- 普通伤害和穿透伤害**交替**：
  - 第1段：普通（0.2×ATK，受防御减免）
  - 第2段：穿透（0.2×ATK，无视防御）
  - 第3段：普通
  - 第4段：穿透
  - 第5段：普通
  - 第6段：穿透
- CD: 0

### 实现要点
```javascript
{ name:'冰锥', type:'iceSpike', hits:6, power:0, pierce:0,
  desc:'6段交替普通/穿透共1.2×ATK', cd:0, totalScale:1.2 }
```
- 新技能类型 `iceSpike`
- 自定义函数 `doIceSpike`：循环6次，每次 0.2×ATK
  - 奇数段（1,3,5）：普通伤害，走 DEF 减免
  - 偶数段（2,4,6）：穿透伤害，无视 DEF
- 每段独立暴击判定
- 每段触发天使龟裁决等 on-hit 被动

---

## 二技能：冰盾

- 为一名**友方目标**（包括自己）获得 **1.2×ATK** 护盾
- CD: 3

### 实现要点
```javascript
{ name:'冰盾', type:'shield', hits:1, power:0, pierce:0, shield:0,
  desc:'友方获得1.2×ATK护盾', cd:3, shieldAtkScale:1.2 }
```
- 复用现有 `shield` 类型，新增 `shieldAtkScale` 参数
- 在 `doShield` 中加入：`if (skill.shieldAtkScale) amount += Math.round(caster.atk * skill.shieldAtkScale)`
- 或使用新类型 `iceShield` 走自定义逻辑

---

## 三技能：冰霜

- 对**所有敌人**造成 **1×ATK 穿透伤害**（无视防御）
- CD: 4

### 实现要点
```javascript
{ name:'冰霜', type:'iceFrost', hits:1, power:0, pierce:0,
  desc:'全体敌方1×ATK穿透伤害', cd:4, atkScale:1.0, aoe:true }
```
- 新技能类型 `iceFrost`，或直接用 AOE 穿透
- 自定义函数 `doIceFrost`：遍历所有存活敌方（含随从），每个造成 1×ATK 穿透伤害
- 穿透伤害不受 DEF 减免，但打护盾

## 伤害模拟（寒冰龟 ATK=38，目标DEF=15）

### 一技能：冰锥（6段交替）
| 段数 | 类型 | 伤害计算 | 备注 |
|------|------|----------|------|
| 第1段 | 普通 | 0.2×38=7.6 → 减防后≈5 | DEF减免 |
| 第2段 | 穿透 | 0.2×38=7.6 → 8 | 无视DEF |
| 第3段 | 普通 | ≈5 | |
| 第4段 | 穿透 | ≈8 | |
| 第5段 | 普通 | ≈5 | |
| 第6段 | 穿透 | ≈8 | |
| **合计** | | **≈39** | 约一半无视防御 |

> 对高防目标比纯普通伤害效率更高，交替穿透是核心优势

### 三技能：冰霜（AOE穿透）
- 每个敌人承受 38 穿透伤害（无视DEF）
- 2v2 场景下对2个敌人 = 76 总伤害

## 平衡考量

- **被动ATK-15%持续6回合**：开场控制强，但时间有限，后期衰减
- **对火系+20%增伤**：定向克制熔岩龟和凤凰龟，让寒冰龟成为火系天敌
- **一技能6段交替**：多段触发各种 on-hit 效果（如天使龟裁决），穿透段无视防御
- **二技能冰盾**：纯辅助，1.2×ATK≈46护盾，中规中矩
- **三技能AOE穿透**：全体1×ATK穿透伤害，对高防队伍有效但单体伤害不高

---

## 与其他龟的协同/克制

### 克制
- **熔岩龟/凤凰龟**：被动+20%增伤，天然克制
- **高攻龟**：被动-15%ATK削弱对方输出

### 被克制
- **闪电龟**：多段电击对寒冰龟消耗快
- **天使龟**：裁决被动按当前HP计算，寒冰龟HP中等偏高容易被吃

### 协同
- 配合石头龟：冰盾+石头龟岩石护甲，双重护盾非常肉
- 配合天使龟：6段冰锥每段触发裁决被动，爆发惊人
