# 熔岩龟 (S级) — 技能设计

## 核心定位
变身爆发型。通过造伤和承伤积累怒气，满怒变身火山龟获得巨幅属性提升和全新技能组，攻守兼备。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 290 |
| ATK | 40 |
| DEF | 14 |
| MR | 16 |
| SPD | 8 |
| 暴击 | 25% |

---

## 被动：熔岩之心

- **造成伤害**时，伤害值的 **25%** 转化为怒气
- **受到伤害**时，伤害值的 **20%** 转化为怒气
- 怒气满 **100** 时变身 **火山龟**，持续 **7回合**
- 变身时获得：
  - **+350%ATK** 的最大HP
  - **ATK+20%**
  - **DEF+20%**
  - **MR+20%**
  - 变身瞬间对全体敌人造成 **120%ATK 魔法伤害** + **灼烧**
- 变身后技能组替换为火山龟技能

### 实现要点
- 新资源 `rage`，上限100，初始0
- 在 `doDamage` 中：造伤时 rage += damage×0.25；受伤时 rage += damage×0.20
- rage >= 100 触发变身：
  - maxHP += 3.50×ATK，currentHP += 3.50×ATK
  - push `{type:'atkUp', value:20, turns:7}`, `{type:'defUp', value:20, turns:7}`, `{type:'mrUp', value:20, turns:7}`
  - 对全体敌人造成 1.20×ATK 魔法伤害 + 灼烧
  - 切换技能组为火山龟技能
- 7回合后恢复原始形态和技能组

## 常态技能

### 一技能：熔岩弹

- 造成 **100%ATK + 9%目标当前HP** 的 **魔法伤害**
- 附加 **灼烧** 效果
- CD: **0**

#### 实现要点
```javascript
{ name:'熔岩弹', type:'lavaBolt', power:1.00, bonusHpScale:0.09,
  damageType:'magic', desc:'100%ATK+9%目标HP魔法+灼烧', cd:0 }
```

### 二技能：地裂

- 对全体敌人造成 **80%ATK 魔法伤害**
- 降低全体敌人 **MR-20%**，持续 **3回合**
- CD: **3**

#### 实现要点
```javascript
{ name:'地裂', type:'earthSplit', power:0.80, damageType:'magic',
  target:'allEnemy', desc:'全体80%ATK魔法-20%MR 3回合', cd:3 }
```
- 对全体敌人 push `{type:'mrDown', value:20, turns:3}`

### 三技能：岩浆涌动

- 造成 **150%ATK 魔法伤害**
- 获得 **80%ATK 护盾**
- CD: **4**

#### 实现要点
```javascript
{ name:'岩浆涌动', type:'magmaSurge', power:1.50, damageType:'magic',
  shieldScale:0.80, desc:'150%ATK魔法+80%ATK护盾', cd:4 }
```

## 火山龟技能（变身后）

### 一技能：烈焰重击

- 造成 **140%ATK + 12%自身当前HP** 的 **物理伤害**
- **吸血35%**（回复造成伤害的35%）
- CD: **0**

#### 实现要点
```javascript
{ name:'烈焰重击', type:'flameSmash', power:1.40, bonusSelfHpScale:0.12,
  damageType:'physical', lifesteal:0.35, desc:'140%ATK+12%自身HP物理+35%吸血', cd:0 }
```

### 二技能：熔岩铠甲

- 获得 **120%ATK 护盾**
- **DEF+25%** 和 **MR+25%**，持续 **3回合**
- 回复 **15%已损失HP**
- CD: **3**

#### 实现要点
```javascript
{ name:'熔岩铠甲', type:'lavaArmor', shieldScale:1.20,
  desc:'120%ATK护盾+25%DEF/MR 3回合+15%已损HP回复', cd:3 }
```
- push `{type:'defUp', value:25, turns:3}`, `{type:'mrUp', value:25, turns:3}`
- 回复 (maxHP - currentHP) × 0.15

### 三技能：火山爆发

- 对全体敌人造成 **5段**，每段 **28%ATK 魔法伤害** + **14%ATK 真实伤害** + **3%目标当前HP**（共140%ATK魔法+70%ATK真实+15%HP）
- 附加 **灼烧** 效果
- 自身回复 **15%HP**
- CD: **4**

#### 实现要点
```javascript
{ name:'火山爆发', type:'volcanoEruption', hits:5, power:0.28, truePower:0.14,
  bonusHpScale:0.03, damageType:'magic', target:'allEnemy',
  desc:'全体5段28%ATK魔+14%ATK真+3%HP+灼烧+15%回复', cd:4 }
```
- 每段对全体敌人造成魔法+真实+%HP伤害
- 全部命中后灼烧+自身回复15%maxHP

## 战斗节奏

1. **常态前期**：熔岩弹持续输出+灼烧，地裂削弱敌方魔抗
2. **积攒怒气**：造伤25%+承伤20%稳步积累
3. **变身爆发**：变身瞬间AOE+灼烧，随后7回合高强度输出
4. **变身期间**：烈焰重击吸血续航，熔岩铠甲抗压，火山爆发全体轰炸
5. **变身结束**：回归常态继续积累怒气

---

## 平衡考量

- 常态SPD8极慢，需要时间积累怒气
- 变身后属性大幅提升（+350%ATK的HP，+20%三维），战力飙升
- 烈焰重击含12%自身HP伤害，变身后HP极高时伤害惊人
- 火山爆发5段全体（魔法+真实+%HP）为顶级AOE
- 变身仅持续7回合，结束后回归脆弱常态
- 常态290HP偏低，可能在变身前被集火击杀
