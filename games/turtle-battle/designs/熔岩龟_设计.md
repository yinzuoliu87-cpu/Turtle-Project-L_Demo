# 熔岩龟 (S级) — 技能设计

## 核心定位
变身爆发型。通过造伤和承伤积累怒气，满怒变身火山龟获得巨幅属性提升和全新技能组，攻守兼备。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 290 |
| ATK | 40 |
| DEF | 14 |
| MR | 16 |
| SPD | 8 |
| 暴击 | 25% |

---

## 被动：熔岩之心

- **造成伤害**时，伤害值的 **25%** 转化为怒气
- **受到伤害**时，伤害值的 **20%** 转化为怒气
- 怒气满 **100** 时变身 **火山龟**，持续 **6回合**
- 变身时获得：
  - **+250%ATK** 的最大HP
  - **ATK+20%**
  - **DEF+20%**
  - **MR+20%**
  - 变身瞬间对全体敌人造成 **50%ATK 魔法伤害** + **灼烧**
- 变身后技能组替换为火山龟技能

### 实现要点
- 新资源 `rage`，上限100，初始0
- 在 `doDamage` 中：造伤时 rage += damage×0.25；受伤时 rage += damage×0.20
- rage >= 100 触发变身：
  - maxHP += 2.50×ATK，currentHP += 2.50×ATK
  - push `{type:'atkUp', value:20, turns:6}`, `{type:'defUp', value:20, turns:6}`, `{type:'mrUp', value:20, turns:6}`
  - 对全体敌人造成 0.50×ATK 魔法伤害 + 灼烧
  - 切换技能组为火山龟技能
- 6回合后恢复原始形态和技能组

## 常态技能

### 一技能：熔岩弹

- 造成 **90%ATK + 8%目标当前HP** 的 **魔法伤害**
- 附加 **灼烧** 效果
- CD: **0**

#### 实现要点
```javascript
{ name:'熔岩弹', type:'lavaBolt', power:0.90, bonusHpScale:0.08,
  damageType:'magic', desc:'90%ATK+8%目标HP魔法+灼烧', cd:0 }
```

### 二技能：地裂

- 对全体敌人造成 **60%ATK 魔法伤害**
- 降低全体敌人 **MR-20%**，持续 **3回合**
- CD: **3**

#### 实现要点
```javascript
{ name:'地裂', type:'earthSplit', power:0.60, damageType:'magic',
  target:'allEnemy', desc:'全体60%ATK魔法-20%MR 3回合', cd:3 }
```
- 对全体敌人 push `{type:'mrDown', value:20, turns:3}`

### 三技能：岩浆涌动

- 造成 **150%ATK 魔法伤害**
- 获得 **80%ATK 护盾**
- CD: **4**

#### 实现要点
```javascript
{ name:'岩浆涌动', type:'magmaSurge', power:1.50, damageType:'magic',
  shieldScale:0.80, desc:'150%ATK魔法+80%ATK护盾', cd:4 }
```

## 火山龟技能（变身后）

### 一技能：烈焰重击

- 造成 **130%ATK + 11%自身当前HP** 的 **物理伤害**
- **吸血25%**（回复造成伤害的25%）
- CD: **0**

#### 实现要点
```javascript
{ name:'烈焰重击', type:'flameSmash', power:1.30, bonusSelfHpScale:0.11,
  damageType:'physical', lifesteal:0.25, desc:'130%ATK+11%自身HP物理+25%吸血', cd:0 }
```

### 二技能：熔岩铠甲

- 获得 **90%ATK 护盾**
- **DEF+20%** 和 **MR+20%**，持续 **3回合**
- 回复 **15%已损失HP**
- CD: **3**

#### 实现要点
```javascript
{ name:'熔岩铠甲', type:'lavaArmor', shieldScale:0.90,
  desc:'90%ATK护盾+20%DEF/MR 3回合+15%已损HP回复', cd:3 }
```
- push `{type:'defUp', value:20, turns:3}`, `{type:'mrUp', value:20, turns:3}`
- 回复 (maxHP - currentHP) × 0.15

### 三技能：火山爆发

- 对全体敌人造成 **5段**，每段 **22%ATK 魔法伤害** + **10%ATK 真实伤害** + **3%自身最大HP**（共110%ATK魔法+50%ATK真实+15%HP）
- 附加 **灼烧** 效果
- 回复总伤害的 **15%** 生命值
- CD: **4**

#### 实现要点
```javascript
{ name:'火山爆发', type:'volcanoEruption', hits:5, power:0.22, truePower:0.10,
  bonusSelfHpScale:0.03, damageType:'magic', target:'allEnemy',
  desc:'全体5段22%ATK魔+10%ATK真+3%自身HP+灼烧+回复总伤害15%', cd:4 }
```
- 每段对全体敌人造成魔法+真实+%自身HP伤害
- 全部命中后灼烧+回复总伤害的15%

## 战斗节奏

1. **常态前期**：熔岩弹持续输出+灼烧，地裂削弱敌方魔抗
2. **积攒怒气**：造伤25%+承伤20%稳步积累
3. **变身爆发**：变身瞬间AOE+灼烧，随后6回合高强度输出
4. **变身期间**：烈焰重击吸血续航，熔岩铠甲抗压，火山爆发全体轰炸
5. **变身结束**：回归常态继续积累怒气

---

## 平衡考量

- 常态SPD8极慢，需要时间积累怒气
- 变身后属性大幅提升（+250%ATK的HP，+20%三维），战力飙升
- 烈焰重击含11%自身HP伤害，变身后HP极高时伤害惊人
- 火山爆发5段全体（魔法+真实+%HP）为顶级AOE
- 变身仅持续6回合，结束后回归脆弱常态
- 常态290HP偏低，可能在变身前被集火击杀
