# 凤凰龟 (S级) — 技能设计

## 核心定位
持续灼烧+复活型。通过灼烧和减益持续消耗敌人，死亡后涅槃复活并对全体敌人施加强力debuff。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 330 |
| ATK | 42 |
| DEF | 12 |
| MR | 15 |
| SPD | 14 |
| 暴击 | 25% |

---

## 被动：涅槃

- **首次死亡**时以 **25%HP** 复活（仅触发一次）
- 复活时对全体敌人施加：
  - **灼烧** 4回合
  - **治疗削减** 3回合（-50%治疗效果）

### 实现要点
- 标记 `phoenixRevived` 防止多次触发
- 在 `onDeath` 中拦截，恢复 25%maxHP，设置 `phoenixRevived = true`
- 复活后对全体敌人 push `{type:'burn', turns:4}` 和 `{type:'healReduction', value:50, turns:3}`
- 日志：`🔥凤凰龟 涅槃复活！全体敌人灼烧4回合+治疗削减3回合`

---

## 一技能：灼烧

- 造成 **90%ATK 魔法伤害**
- 附加 **灼烧** 效果，持续 **4回合**
- CD: **0**

### 实现要点
```javascript
{ name:'灼烧', type:'burnAttack', power:0.90, damageType:'magic',
  desc:'90%ATK魔法+灼烧4回合', cd:0 }
```
- 命中后对目标 push `{type:'burn', turns:4}`

---

## 二技能：熔岩盾

- 获得 **100%ATK 护盾**，持续 **4回合**
- 护盾存在期间，受到攻击时 **反击** 攻击者，造成 **16%ATK 魔法伤害**
- CD: **4**

### 实现要点
```javascript
{ name:'熔岩盾', type:'lavaShield', shieldScale:1.00,
  desc:'100%ATK护盾4回合+反击16%ATK魔法', cd:4 }
```
- push `{type:'shield', value: 1.00×ATK, turns:4, retaliate:true, retaliateScale:0.16, retaliateDamageType:'magic'}`
- 反击在 `onHit` 中触发

---

## 三技能：烫伤

- 破坏目标 **50%护盾**
- 造成 **70%ATK 魔法伤害**
- 降低目标 **ATK-15%** 和 **DEF-15%**，持续 **3回合**
- CD: **3**

### 实现要点
```javascript
{ name:'烫伤', type:'scald', power:0.70, damageType:'magic',
  shieldBreak:0.50, desc:'破50%护盾+70%ATK魔法-15%ATK-15%DEF 3回合', cd:3 }
```
- 先移除目标50%当前护盾值
- 造成伤害后 push `{type:'atkDown', value:15, turns:3}` 和 `{type:'defDown', value:15, turns:3}`

---

## 战斗节奏

1. **常规回合**：灼烧维持持续魔法伤害+灼烧DOT
2. **防御回合**：熔岩盾提供护盾+反击
3. **对护盾敌人**：烫伤破盾+削弱
4. **被击杀时**：涅槃复活，全体灼烧+治疗削减，翻盘能力强

---

## 平衡考量

- 涅槃仅触发一次，等效额外25%HP+全体debuff
- 灼烧DOT提供持续伤害，但单次爆发不高
- 熔岩盾反击伤害较低（16%ATK），主要提供生存能力
- 烫伤的破盾+双降为功能性技能，对坦克有效
- 治疗削减-50%克制回复型阵容
- SPD14偏慢，适合中后期发力
