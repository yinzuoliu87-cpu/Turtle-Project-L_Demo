# 钻石龟 (B级) — 技能设计

## 核心定位
纯坦克型。被动强化全队护甲/魔抗并提供固定伤害减免，技能基于DEF/MR缩放，攻守一体。

## 被动：钻石结构

- 全队所有 **护甲/魔抗加成**（buff提供的）额外 **+50%** 效果
- 每段受到伤害时 **固定减免 20%×DEF**（真实伤害除外）
- 例：DEF21时，每段伤害固定减免 21×0.20 = 4.2点

### 实现要点
- 在 buff 计算中，检测队伍中是否有钻石龟存活，若有则所有 `defUp`/`mrUp` buff 的 value ×1.5
- 在 `doDamage` 中，若目标为钻石龟且伤害类型非真实，最终伤害 -= DEF×0.20（最低为1）
- 固定减免在百分比减免之后结算

## 一技能：钻石切割

- 造成 **(70%ATK + 70%DEF + 70%MR) 物理伤害**
- CD: **0**

### 实现要点
```javascript
{ name:'钻石切割', type:'diamondCut', damageType:'physical',
  desc:'70%ATK+70%DEF+70%MR物理', cd:0 }
```
- 伤害 = ATK×0.70 + DEF×0.70 + MR×0.70
- 基础面板：38×0.70 + 21×0.70 + 18×0.70 = 26.6 + 14.7 + 12.6 = 53.9

## 二技能：坚不可摧

- 获得 **20%HP 护盾**，持续 **3回合**
- 获得 **+20%ATK护甲**（即 ATK×0.20 点DEF）和 **+20%ATK魔抗**（即 ATK×0.20 点MR），持续 **3回合**
- CD: **3**

### 实现要点
```javascript
{ name:'坚不可摧', type:'indestructible',
  desc:'20%HP护盾3回合+20%ATK护甲+20%ATK魔抗 3回合', cd:3 }
```
- 护盾值 = maxHP × 0.20
- push `{type:'defUp', flat: ATK×0.20, turns:3}`, `{type:'mrUp', flat: ATK×0.20, turns:3}`

## 三技能：碰撞

- 造成 **(80%ATK + 90%DEF + 90%MR + 8%自身HP) 物理伤害**
- 对同一目标 **累计命中2次** 后，施加 **眩晕**
- CD: **3**

### 实现要点
```javascript
{ name:'碰撞', type:'collision', damageType:'physical',
  desc:'80%ATK+90%DEF+90%MR+8%HP物理,累计2次眩晕', cd:3 }
```
- 伤害 = ATK×0.80 + DEF×0.90 + MR×0.90 + currentHP×0.08
- 基础面板：38×0.80 + 21×0.90 + 18×0.90 + 361×0.08 = 30.4 + 18.9 + 16.2 + 28.88 = 94.38
- 记录对每个目标的碰撞命中次数 `collisionCount`，达到2次时 push `{type:'stun', turns:1}`

---

## 战斗节奏

1. **回合1**：坚不可摧获得护盾和双抗提升
2. **常规回合**：钻石切割稳定输出（基于双抗的高基础伤害）
3. **每3回合**：碰撞高伤害+眩晕控制
4. 被动固定减免使其持续抗压

---

## 平衡考量

- 被动全队护甲/魔抗加成+50%，与增益型队友协同极强
- 固定减免20%×DEF对多段攻击特别有效（每段都减）
- DEF/MR缩放技能使堆防也能提升伤害
- 碰撞累计2次眩晕提供可靠控制
- ATK38偏低档，纯ATK倍率伤害低
- SPD8极慢，依赖队友先手
- 真实伤害无视固定减免，是其克星
