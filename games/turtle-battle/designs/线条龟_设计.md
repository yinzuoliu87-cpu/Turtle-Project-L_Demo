# 线条龟 (A级) — 技能设计

## 核心定位
叠层增伤型。通过墨迹标记持续放大全队对目标的伤害，满层后引爆造成爆发真实伤害。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 332 |
| ATK | 46 |
| DEF | 10 |
| MR | 11 |
| SPD | 16 |
| 暴击 | 25% |

---

## 被动：墨迹标记

- 攻击目标时叠加 **墨迹** 层数，上限 **5层**
- 每层墨迹使目标受到的 **所有伤害+5%**（满层+25%）
- 墨迹对所有来源的伤害均生效，包括队友
- 「连笔」连接的两个敌人共享墨迹层数
- 「画龙点睛」引爆目标墨迹，造成大量真实伤害

### 实现要点
- 新 debuff 类型 `inkMark`，`{type:'inkMark', stacks:1, maxStacks:5}`
- 在 `doDamage` 中检测目标身上的 `inkMark` 层数，最终伤害 x(1 + stacks x 0.05)
- 墨迹层数在目标头顶以图标+数字显示

---

## 一技能：素描

- **3段伤害**，每段 **45%ATK 物理伤害**（共135%ATK）
- 每段命中叠加 **1层墨迹**（单次最多叠3层）
- CD: **0**

### 实现要点
```javascript
{ name:'素描', type:'lineSketch', hits:3, power:0, pierce:0,
  desc:'3段45%ATK物理+每段叠1层墨迹', cd:0, atkScale:0.45 }
```
- 每段 hit 后对目标 push/increment `inkMark`

---

## 二技能：连笔

- 连接 **2个敌人**，持续 **3回合**
- 对两个目标各造成 **80%ATK 物理伤害**，各叠加 **1层墨迹**
- 连接期间，一方受到的伤害 **30%** 以 **真实伤害** 传递给另一方
- 连接双方的 **墨迹层数共享**（一方获得墨迹时另一方也同步）
- CD: **3**

### 实现要点
```javascript
{ name:'连笔', type:'lineLink', hits:1, power:0, pierce:0,
  desc:'连接2敌3回合,80%ATK物理+叠墨迹+伤害传递30%真实+墨迹共享', cd:3,
  atkScale:0.8, duration:3, transferPct:30 }
```
- 新状态 `inkLinked`：记录连接对象ID，持续3回合
- 伤害传递在 `doDamage` 结算后触发，传递部分为真实伤害不受防御减免
- 一方获得墨迹时，另一方也同步获得

---

## 三技能：画龙点睛

- 引爆目标身上的全部 **墨迹**
- 造成 **70%ATK 物理伤害** + 每层墨迹额外 **45%ATK 真实伤害**
- 满5层时：70%ATK物理 + 225%ATK真实
- 引爆后墨迹层数清零
- CD: **4**

### 实现要点
```javascript
{ name:'画龙点睛', type:'lineFinish', hits:1, power:0, pierce:0,
  desc:'70%ATK物理+每层墨迹45%ATK真实,引爆清零', cd:4,
  baseScale:0.7, perStackScale:0.45 }
```
- 读取目标 `inkMark` 层数，额外造成 stacks x 0.45 x ATK 真实伤害
- 引爆后移除目标 `inkMark` debuff

---

## 战斗节奏

1. **回合1-2**：素描快速叠层（每回合3层，2回合可满5层）
2. **回合3**：画龙点睛引爆满层墨迹，爆发真实伤害
3. **适时使用连笔**：对双目标同时叠墨迹并传递伤害，配合队友集火
4. 循环：叠层→引爆→叠层→引爆

---

## 平衡考量

- 满层墨迹的+25%增伤对全队生效，辅助价值高
- 画龙点睛满层爆发（70%物理+225%真实）需要至少2回合准备
- 连笔的伤害传递和墨迹共享提供多目标压制能力
- SPD16中等偏快，适合先手叠层
- 无护盾/回复能力，生存依赖队友保护
