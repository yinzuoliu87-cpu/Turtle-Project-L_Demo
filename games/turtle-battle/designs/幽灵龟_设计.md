# 幽灵龟 (B级) — 技能设计

## 核心定位
真实伤害+死后诅咒型。以高比例真实伤害无视防御，死亡后诅咒全体敌人持续掉血，生死皆威胁。

---

## 被动：怨灵

- **死亡时**诅咒全体敌人，持续 **3回合**
- 每回合造成 **9%目标最大HP** 的 **真实伤害**
- 3回合共计 27%目标最大HP 真实伤害

### 实现要点
- 在 `onDeath` 中触发，对全体存活敌人 push `{type:'curse', hpPercent:0.09, turns:3, damageType:'true'}`
- 每回合结算时，诅咒目标受到 maxHP×0.09 真实伤害
- 日志：`👻幽灵龟 怨灵：全体敌人被诅咒3回合`

---

## 一技能：幽魂触碰

- 造成 **40%ATK 物理伤害** + **90%ATK 真实伤害**
- CD: **0**

### 实现要点
```javascript
{ name:'幽魂触碰', type:'ghostTouch', power:0.40, truePower:0.90,
  damageType:'physical', desc:'40%ATK物理+90%ATK真实', cd:0 }
```
- 物理部分受防御减免，真实部分无视防御
- 真实伤害占比极高（90%ATK vs 40%ATK）

---

## 二技能：虚化

- 获得 **60%闪避率**，持续 **2回合**
- 获得 **90%ATK 护盾**
- CD: **3**

### 实现要点
```javascript
{ name:'虚化', type:'phaseShift', dodgeRate:0.60, dodgeTurns:2,
  shieldScale:0.90, desc:'60%闪避2回合+90%ATK护盾', cd:3 }
```
- push `{type:'dodge', value:60, turns:2}` 和护盾
- 闪避判定在 `doDamage` 前进行，闪避时伤害为0

---

## 三技能：灵魂风暴

- **5段伤害**，每段 **35%ATK 真实伤害**（共175%ATK真实）
- 附加 **9%目标最大HP诅咒**，持续 **3回合**
- CD: **5**

### 实现要点
```javascript
{ name:'灵魂风暴', type:'soulStorm', hits:5, truePower:0.35,
  damageType:'true', desc:'5段35%ATK真实+9%目标HP诅咒3回合', cd:5 }
```
- 5段纯真实伤害，无视一切防御
- 命中后 push `{type:'curse', hpPercent:0.09, turns:3, damageType:'true'}`

---

## 战斗节奏

1. **常规回合**：幽魂触碰持续输出（40%物理+90%真实），对坦克极为有效
2. **受到集火时**：虚化闪避+护盾抗压
3. **爆发回合**：灵魂风暴175%ATK真实+诅咒，毁灭性单体伤害
4. **被击杀时**：怨灵诅咒全体3回合，9%maxHP/回合，死后持续输出

---

## 平衡考量

- 幽魂触碰90%ATK真实伤害为常驻无视防御输出，克制高防坦克
- 灵魂风暴175%ATK纯真实+诅咒，CD5回合的超强单体
- 怨灵死后诅咒27%maxHP真实（3回合），使对手不愿击杀
- DEF10/MR10极脆，双抗最低档
- 虚化60%闪避提供不稳定但高效的生存手段
- HP319偏低，没有虚化时容易被秒
- 真实伤害比例极高，不受对方防御和增防影响
