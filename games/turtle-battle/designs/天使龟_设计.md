# 天使龟 (A级) — 技能设计

## 核心定位
裁决者/半辅助。对高稀有度目标有额外克制，兼具护盾辅助能力。被动让每次攻击都附带基于目标当前生命值的额外伤害，越肉的敌人越疼。

---

## 被动：裁决

- 每段攻击额外附带目标 **当前HP 9%** 的 **普通伤害**
- 适用于所有造成伤害的技能（一技能、三技能的每一段均触发）
- 该额外伤害受目标防御减免（普通伤害）
- 该额外伤害可以暴击（跟随本次攻击的暴击判定）
- 三技能"平等"对S级以上目标的回复效果也计入被动造成的伤害

### 实现要点
- 在 `doDamage` 或 `triggerOnHitEffects` 中检测攻击者被动类型 `judgement`
- 每段 hit 后额外计算 `target.hp * 0.09` 的普通伤害（走正常防御减免流程）
- 浮动数字显示为独立的金色伤害数字，如 `⚖12`
- 日志标记为 `<span class="log-passive">⚖裁决 XX</span>`

---

## 一技能：裁决

- 对目标打击 **4次**，共计 **1.2×ATK** 普通伤害
- 每次命中 = 0.3×ATK，每次均触发被动（共触发4次被动）
- CD: 0

### 实现要点
```javascript
{ name:'裁决', type:'physical', hits:4, power:0, pierce:0,
  desc:'4次共1.2×ATK普通伤害，每段触发被动裁决',
  cd:0, atkScale:1.2 }
```
- 4段攻击，每段独立触发被动的 9% 当前HP 伤害
- 由于是4段，被动总共触发4次，对高HP目标伤害可观

---

## 二技能：祝福

- 对一名友方目标施加：
  - **1.2×ATK** 的护盾，持续 **4回合**
  - **防御+0.12×ATK**，持续 **4回合**
- CD: 4

### 实现要点
```javascript
{ name:'祝福', type:'angelBless', hits:1, power:0, pierce:0,
  desc:'友方获得1.2×ATK护盾4回合 + 防御+0.12×ATK 4回合',
  cd:4, shieldScale:1.2, shieldTurns:4, defBoostScale:0.9, defBoostTurns:4 }
```
- 新技能类型 `angelBless`
- 护盾值 = `Math.round(caster.atk * 1.2)`
- 防御增加值 = `Math.round(caster.atk * 0.9)`
- 目标选择：友方（包括自己），优先给血量最低/护盾最少的队友
- AI 逻辑：优先给血量比最低的友方施放

---

## 三技能：平等

- 对目标造成 **两段伤害**：
  - 第一段：**0.6×ATK 普通伤害**（受防御减免）
  - 第二段：**0.6×ATK 穿透伤害**（无视防御）
- 每段均触发被动裁决（共2次被动）
- **特殊效果**：如果目标是 **S级或SS级** 的龟：
  - 本次攻击 **必定暴击**（两段均暴击）
  - 回复本技能造成的 **所有伤害的 10%** 为自身HP（包括被动裁决造成的伤害）
- CD: 5

### 实现要点
```javascript
{ name:'平等', type:'angelEquality', hits:2, power:0, pierce:0,
  desc:'0.6×ATK普通+0.6×ATK穿透，S级以上必暴击并回复总伤10%HP',
  cd:5, normalScale:0.6, pierceScale:0.6,
  antiHighRarity:['S','SS'], forceCrit:true, healPctOfDmg:10 }
```
- 新技能类型 `angelEquality`
- 第一段走普通伤害流程（被动触发）
- 第二段走穿透伤害流程（被动也触发，被动伤害仍为普通伤害）
- 判定目标 rarity 是否在 `['S','SS']` 中：
  - 是 → 强制 `isCrit = true`，记录总伤害（含被动），回复 10%
  - 否 → 正常暴击判定，无回复
- 回复的浮动数字：绿色 `+XX`

## 伤害模拟（天使龟 ATK=36，目标HP=300，DEF=15）

### 一技能：裁决（4段）
| 段数 | 技能伤害 | 被动(9%当前HP) | 备注 |
|------|----------|----------------|------|
| 第1段 | 0.3×36=10.8 → ~11 | 300×9%=27 → 减防后~17 | 目标HP≈272 |
| 第2段 | ~11 | 272×9%≈24 → ~15 | 目标HP≈246 |
| 第3段 | ~11 | 246×9%≈22 → ~14 | 目标HP≈221 |
| 第4段 | ~11 | 221×9%≈20 → ~13 | 目标HP≈197 |
| **合计** | **~44** | **~59** | **总伤害≈103** |

> 被动伤害占比超过50%，对高HP目标非常有效

### 三技能：平等（vs S级目标，必暴击）
| 段数 | 技能伤害 | 被动 | 暴击 |
|------|----------|------|------|
| 普通段 | 0.6×36≈22 → 减防后~14 | 9%当前HP | 必暴击×1.5 |
| 穿透段 | 0.6×36≈22（无视防御） | 9%当前HP | 必暴击×1.5 |
| **回复** | 总伤害×10% | | |

## 平衡考量

- **被动9%当前HP**：前期对满血目标伤害高，后期递减（当前HP越低伤害越低），自带平衡机制
- **一技能4段触发**：多段 = 多次被动，是天使龟的主要输出手段
- **二技能祝福**：纯辅助，给队友提供大量护盾+防御，让天使龟不只是输出
- **三技能平等**：对S/SS级龟的克制机制，必暴击+回复让天使龟在对抗高稀有度时有优势
- **弱点**：自身HP和防御中等，被集火容易倒；被动对低HP目标伤害递减

---

## 与其他龟的协同/克制

### 克制
- 高HP坦克（石头龟、缩头乌龟）：被动按比例吃HP，越肉越疼
- S/SS级龟：三技能"平等"必暴击+回复

### 被克制
- 高爆发龟（无头龟、猎人龟）：天使龟自身不够肉，容易被秒
- 闪电龟：多段电击消耗快

### 协同
- 配合高防队友：二技能"祝福"进一步加强防御
- 配合缩头乌龟：祝福给缩头乌龟叠盾+防御，非常厚
