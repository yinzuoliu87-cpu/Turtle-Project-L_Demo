# 天使龟 (B级) — 技能设计

## 核心定位
裁决者/半辅助。被动让每次攻击都附带基于目标当前生命值的额外魔法伤害，越肉的敌人越疼。兼具护盾辅助能力和对高稀有度目标的克制。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 340 |
| ATK | 41 |
| DEF | 13 |
| MR | 16 |
| SPD | 11 |
| 暴击 | 25% |

---

## 被动：审判

- 每段攻击额外附带目标 **当前HP 11%** 的 **魔法伤害**
- 适用于所有造成伤害的技能（一技能、三技能的每一段均触发）
- 该额外伤害受目标魔抗减免（魔法伤害）
- 该额外伤害可以暴击（跟随本次攻击的暴击判定）
- 三技能"平等"对S级以上目标的回复效果也计入被动造成的伤害

### 实现要点
- 在 `doDamage` 或 `triggerOnHitEffects` 中检测攻击者被动类型 `judgement`
- 每段 hit 后额外计算 `target.hp * 0.11` 的魔法伤害（走魔抗减免流程）
- 浮动数字显示为独立的金色伤害数字，如 `⚖12`
- 日志标记为 `<span class="log-passive">⚖审判 XX</span>`

---

## 一技能：裁决

- 对目标打击 **4段**，每段 **35%×ATK** 物理伤害
- 共 **140%×ATK** 物理伤害
- 每次命中均触发被动（共触发4次被动）
- CD: 0

### 实现要点
```javascript
{ name:'裁决', type:'physical', hits:4, power:0, pierce:0,
  desc:'4段35%ATK物理伤害，每段触发被动审判',
  cd:0, atkScale:0.35 }
```
- 4段攻击，每段独立触发被动的 11% 当前HP 魔法伤害
- 由于是4段，被动总共触发4次，对高HP目标伤害可观

---

## 二技能：祝福

- 对一名友方目标施加：
  - **120%×ATK** 的护盾，持续 **3回合**
  - **护甲+15%×ATK**，持续 **4回合**
  - **魔抗+15%×ATK**，持续 **4回合**
- CD: 3

### 实现要点
```javascript
{ name:'祝福', type:'angelBless', hits:1, power:0, pierce:0,
  desc:'友方获得120%ATK护盾3回合 + 15%ATK护甲/魔抗4回合',
  cd:3, shieldScale:1.2, shieldTurns:3, defBoostScale:0.15, defBoostTurns:4 }
```
- 技能类型 `angelBless`
- 护盾值 = `Math.round(caster.atk * 1.2)`
- 护甲增加值 = `Math.round(caster.atk * 0.15)`
- 魔抗增加值 = `Math.round(caster.atk * 0.15)`
- 目标选择：友方（包括自己），优先给血量最低/护盾最少的队友

---

## 三技能：平等

- 对目标造成 **两段伤害**：
  - 第一段：**80%×ATK 魔法伤害**（受魔抗减免）
  - 第二段：**80%×ATK 真实伤害**（无视防御和魔抗）
- 每段均触发被动审判（共2次被动）
- **特殊效果**：如果目标是 **A级、S级或SS级** 的龟：
  - 本次攻击 **必定暴击**（两段均暴击）
  - 回复本技能造成的 **所有伤害的 10%** 为自身HP（包括被动审判造成的伤害）
- CD: 5

### 实现要点
```javascript
{ name:'平等', type:'angelEquality', hits:2, power:0, pierce:0,
  desc:'80%ATK魔法+80%ATK真实，A/S/SS级必暴击+10%伤害吸血',
  cd:5, normalScale:0.8, pierceScale:0.8,
  antiHighRarity:['A','S','SS'], forceCrit:true, healPctOfDmg:10 }
```
- 技能类型 `angelEquality`
- 第一段走魔法伤害流程（被动触发）
- 第二段走真实伤害流程（被动也触发，被动伤害仍为魔法伤害）
- 判定目标 rarity 是否在 `['A','S','SS']` 中：
  - 是 -> 强制 `isCrit = true`，记录总伤害（含被动），回复 10%
  - 否 -> 正常暴击判定，无回复
- 回复的浮动数字：绿色 `+XX`

## 平衡考量

- **被动11%当前HP魔法伤害**：前期对满血目标伤害高，后期递减（当前HP越低伤害越低），自带平衡机制
- **一技能4段触发**：多段 = 多次被动，是天使龟的主要输出手段
- **二技能祝福**：纯辅助，给队友提供护盾+护甲+魔抗，让天使龟不只是输出
- **三技能平等**：魔法+真实双段伤害，对A/S/SS级龟的克制机制，必暴击+吸血让天使龟在对抗高稀有度时有优势
- **弱点**：B级基础属性中等，被集火容易倒；被动对低HP目标伤害递减

---

## 与其他龟的协同/克制

### 克制
- 高HP坦克（石头龟、缩头乌龟）：被动按比例吃HP，越肉越疼
- A/S/SS级龟：三技能"平等"必暴击+吸血

### 被克制
- 高爆发龟（无头龟、猎人龟）：天使龟自身不够肉，容易被秒
- 闪电龟：多段电击消耗快

### 协同
- 配合高防队友：二技能"祝福"进一步加强护甲和魔抗
- 配合缩头乌龟：祝福给缩头乌龟叠盾+防御，非常厚
