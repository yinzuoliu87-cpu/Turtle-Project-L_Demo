# 赛博龟 (S级) — 技能设计

## 核心定位
远程输出 + 死后转化型。通过浮游炮积累战场压力，阵亡后转化为机甲继续战斗。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 360 |
| ATK | 47 |
| DEF | 14 |
| MR | 13 |
| SPD | 16 |
| 暴击 | 25% |

---

## 被动：浮游炮

### 浮游炮生成
- 每回合自动生成 **1个浮游炮**
- 三技能可额外获得 1个
- **最多持有10个**
- 浮游炮不可选中、无生命值、不占位

### 浮游炮打击
- 每个浮游炮每回合对随机一个敌人造成 **14%×ATK 物理伤害**
- 每回合自动发射（回合结束时触发）

### 阵亡转化：机甲
- 赛博龟阵亡时，所有浮游炮组装成 **机甲**
- 机甲属性：
  - HP = **30 × 浮游炮数量**（例如10个 = 300HP）
  - ATK = **5 × 浮游炮数量**（例如10个 = 50ATK）
  - DEF = **0**
  - 暴击等 = 默认值
- 机甲**替代赛博龟的行动回合**
- 机甲**自动随机攻击**一个敌人，造成 **150%×ATK 物理伤害**（不手动操作）
- 机甲阵亡 = 真正退场
- 如果浮游炮数量为0，不生成机甲

### UI显示
- 赛博龟卡片上显示浮游炮数量
- 机甲替代赛博龟卡片显示（名字改为"机甲"，新头像）

---

## 一技能：激光枪

- 对单体目标打击 **5段**
- 每段造成 **15%×ATK + 2.4%目标HP** 物理伤害
- 共 **75%×ATK + 12%目标HP**
- CD: 0

```javascript
{ name:'激光枪', type:'physical', hits:5, power:0, pierce:0, cd:0,
  atkScale:0.15, hpPct:2.4,
  brief:'赛博龟发射激光5段，共75%ATK+12%目标HP物理伤害' }
```

---

## 二技能：增益

- 自身攻击力 **+27%** 持续 **4回合**
- CD: 3

```javascript
{ name:'增益', type:'heal', hits:1, power:0, pierce:0, cd:3,
  selfAtkUpPct:{pct:27, turns:4},
  brief:'赛博龟提升自身攻击力27%，持续4回合' }
```

---

## 三技能：部署浮游炮

- 立即获得 **1个浮游炮**（不受每回合自动生成影响）
- CD: 2

```javascript
{ name:'部署', type:'cyberDeploy', hits:1, power:0, pierce:0, cd:2,
  brief:'赛博龟立即部署1个浮游炮（上限10）' }
```

## 实现要点

### 浮游炮数据
```javascript
f._drones = [];  // 每个元素 = { age: 0 }
f._droneCount    // getter: f._drones.length
```

### 回合处理 (processBuffs / beginTurn)
1. 赛博龟存活：`f._drones.push({ age:0 })`（不超过10个）
2. 所有浮游炮对随机敌人造成 14%×ATK 物理伤害
3. 更新卡片上的浮游炮计数

### 阵亡处理 (checkDeaths)
1. 检查 `f.passive.type === 'cyberDrone'` 且 `f._drones.length > 0`
2. 创建机甲 fighter（类似缩头乌龟的随从）
3. 机甲替代赛博龟在 `allFighters` 中的位置
4. 机甲不受玩家控制，每回合自动攻击

### 机甲行动
- 在 `showActionPanel` 中检测：如果当前行动的是机甲（`f._isMech`），自动 AI 出招
- 造成 150%×ATK 物理伤害

---

## 平衡考量

- 前期积累浮游炮 = 持续骚扰伤害，每回合每个炮14%ATK
- 10个浮游炮同时输出 = 10 × 14%ATK = 140%ATK 每回合额外伤害
- 机甲 HP = 300（10炮），ATK = 50，无防御，150%ATK自动攻击 = 75伤害/回合
- 如果赛博龟快速被击杀，机甲只有少量浮游炮 -> 很弱
- 二技能增益先buff再积累浮游炮 -> 浮游炮伤害也会受buff影响（用当前ATK）
- 三技能加速积累 -> 但占用了一次攻击机会
