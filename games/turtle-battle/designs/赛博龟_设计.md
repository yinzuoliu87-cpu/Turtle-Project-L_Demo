# 赛博龟 (S级) — 技能设计

## 核心定位
远程输出 + 死后转化型。通过浮游炮积累战场压力，阵亡后转化为机甲继续战斗。

---

## 被动：浮游炮系统

### 浮游炮生成
- 每回合自动生成 **1个浮游炮**
- 三技能可额外获得 1个
- **最多持有10个**
- 浮游炮不可选中、无生命值、不占位

### 浮游炮打击
- 浮游炮存在 **5回合后自动发射**
- 每个浮游炮对随机一个敌人造成 **30%×ATK 普通伤害**
- 发射后消耗（回合结束时检查并触发）

### 阵亡转化：机甲
- 赛博龟阵亡时，所有浮游炮组装成 **机甲**
- 机甲属性：
  - HP = **30 × 浮游炮数量**（例如10个 = 300HP）
  - ATK = **5 × 浮游炮数量**（例如10个 = 50ATK）
  - DEF = **0**
  - 暴击/穿甲等 = 默认值（8%暴击，0穿甲）
- 机甲**替代赛博龟的行动回合**
- 机甲**自动随机攻击**一个敌人，造成 **100%×ATK 普通伤害**（不手动操作）
- 机甲阵亡 = 真正退场
- 如果浮游炮数量为0，不生成机甲

### UI显示
- 赛博龟卡片上显示浮游炮数量：`🛸×3`
- 每个浮游炮带倒计时（剩余几回合发射）
- 机甲替代赛博龟卡片显示（名字改为"机甲"，新头像）

## 一技能：激光枪

- 对单体目标打击 **5段**
- 每段造成 **16%×ATK + 2.4%目标最大HP** 普通伤害
- 共 **80%×ATK + 12%目标最大HP**
- CD: 0

```javascript
{ name:'激光枪', type:'physical', hits:5, power:0, pierce:0, cd:0,
  atkScale:0.16, hpPct:2.4,
  brief:'赛博龟发射激光5段，共{N:0.16*ATK*5}普通伤害+12%目标最大HP',
  detail:'赛博龟对单体目标发射激光5段，每段造成 16%×(攻击力={ATK}) = {N:0.16*ATK} + 2.4%目标最大HP 普通伤害，共 80%×(攻击力={ATK}) = {N:0.16*ATK*5} + 12%目标最大HP 普通伤害。'
}
```

## 实现要点

### 浮游炮数据
```javascript
f._drones = [];  // 每个元素 = { age: 0 }，age 每回合+1
f._droneCount    // getter: f._drones.length
```

### 回合处理 (processBuffs / beginTurn)
1. 赛博龟存活：`f._drones.push({ age:0 })`（不超过10个）
2. 所有浮游炮 `age++`
3. `age >= 5` 的浮游炮发射（对随机敌人造成伤害）并移除
4. 更新卡片上的浮游炮计数

### 阵亡处理 (checkDeaths)
1. 检查 `f.passive.type === 'cyberDrone'` 且 `f._drones.length > 0`
2. 创建机甲 fighter（类似缩头乌龟的随从）
3. 机甲替代赛博龟在 `allFighters` 中的位置
4. 机甲不受玩家控制，每回合自动攻击

### 机甲行动
- 在 `showActionPanel` 中检测：如果当前行动的是机甲（`f._isMech`），自动 AI 出招
- 造成 100%×ATK 普通伤害

### UI
- 浮游炮显示在 status-icons 区域：`🛸×3 (最老:3回合)`
- 机甲替换卡片内容（emoji改为🤖，名字改为"机甲"）

## 平衡考量

- 前期积累浮游炮 = 延迟输出，不如直接输出的龟
- 10个浮游炮同时发射 = 10 × 0.3×ATK = 3.0×ATK 一波爆发
- 机甲 HP = 300（10炮），ATK = 50，无防御 → 脆但有输出
- 如果赛博龟快速被击杀，机甲只有少量浮游炮 → 很弱
- 二技能增益先buff再积累浮游炮 → 浮游炮伤害也会受buff影响（用发射时的ATK）
- 三技能加速积累 → 但占用了一次攻击机会
