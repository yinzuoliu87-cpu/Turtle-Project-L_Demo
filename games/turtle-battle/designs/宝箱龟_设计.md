# 宝箱龟 (S级) — 技能设计

## 核心定位
装备获取型。通过造成伤害积累财宝值，达到阈值自动抽取装备强化自身，越打越强。

---

## 属性
| 属性 | 值 |
|------|-----|
| HP | 345 |
| ATK | 40 |
| DEF | 16 |
| MR | 14 |
| SPD | 10 |
| 暴击 | 25% |

---

## 被动：藏宝图

- 造成伤害时积累 **财宝值**（等于造成的伤害值）
- 达到阈值时自动抽取装备：
  - **50** 财宝值：第1件装备
  - **120** 财宝值：第2件装备
  - **220** 财宝值：第3件装备
  - **350** 财宝值：第4件装备
  - **500** 财宝值：第5件装备
- 装备池分3档：
  - **基础池**（6件装备）
  - **进阶池**（6件装备）
  - **传说池**（3件装备）
- 抽到的装备自动装备，提供被动属性/效果加成

### 实现要点
- 新资源 `treasureValue`，无上限
- 阈值数组 `[50, 120, 220, 350, 500]`
- 在 `doDamage` 中累加 treasureValue
- 达到阈值时从装备池随机抽取，装备池和具体装备效果另行定义
- UI显示当前财宝值和已获装备

---

## 一技能：宝箱砸击

- **4段伤害**，共 **160%ATK 物理伤害**（每段40%ATK）
- 已装备的装备提供 **附加效果**（根据装备类型不同）
- CD: **0**

### 实现要点
```javascript
{ name:'宝箱砸击', type:'chestSmash', hits:4, power:0.40, damageType:'physical',
  desc:'4段160%ATK物理+装备附加效果', cd:0 }
```
- 每段 hit 后检查已装备的装备，触发对应附加效果

---

## 二技能：清点财宝

- 回复 **15%HP**
- 获得 **70%ATK 护盾**
- 每拥有 **100财宝值**，护盾额外 **+14%**
- CD: **4**

### 实现要点
```javascript
{ name:'清点财宝', type:'countTreasure',
  desc:'15%HP回复+70%ATK护盾+每100财宝值+14%', cd:4 }
```
- 回复 maxHP×0.15
- 护盾值 = ATK × (0.70 + Math.floor(treasureValue/100) × 0.14)

---

## 三技能：财宝风暴

- 对全体敌人造成 **10段**，每段 **9%ATK 物理伤害** + **4%ATK 真实伤害**（共90%ATK物理+40%ATK真实）
- CD: **4**

### 实现要点
```javascript
{ name:'财宝风暴', type:'treasureStorm', hits:10, power:0.09, truePower:0.04,
  damageType:'physical', target:'allEnemy',
  desc:'全体10段9%ATK物理+4%ATK真实', cd:4 }
```

---

## 战斗节奏

1. **前期**：宝箱砸击持续输出积累财宝值
2. **50/120财宝值**：获得前两件装备，战力提升
3. **中期**：清点财宝回复+护盾维持生存
4. **后期**：装备越多越强，财宝风暴全体轰炸
5. 战斗越久，装备越多，越后期越强

---

## 平衡考量

- 装备机制使其越打越强，属于后期发力型
- 前期ATK40偏低，财宝积累需要时间
- 宝箱砸击160%ATK+装备附加效果，后期主力输出可观
- 清点财宝的护盾随财宝值增长，后期护盾极厚
- 财宝风暴10段全体提供大量财宝值积累
- SPD10慢速，但后期装备加成可弥补
- 装备随机性带来不确定性，每局体验不同
