# 龟投（Turtle Prediction）— 前端设计需求文档

> 给前端开发工程师的完整设计说明，涵盖所有页面、组件、交互、动效、数据结构与结算逻辑。

---

## 一、项目概述

**产品名称：** 龟投
**定位：** 热点预测 + 社区对战平台
**核心玩法：** 用户围绕热点话题用「龟币」下注预测，同时可进入「开战广场」阵营对抗，评论 + 点赞驱动龟势，最终结算收益。

**页面清单：**

| 页面 | 路径 | 说明 |
|------|------|------|
| 主站首页 | `index.html` | 8 个 Tab（全部/世界杯/热点/科技/AI/娱乐/论坛/开战广场） |
| 对战详情页 | `battle-event.html` | 单场对战深度交互（龟势对抗 + 实时评论） |
| 宠物中心 | `pet.html` | 宠物抽奖、对话、形象切换、披风切换 |

## 二、设计 Token（全局变量）

> 暂只定义暗/亮两套主题变量骨架，具体字体、字号、颜色细节由设计师后续补充。

### 2.1 亮色主题（主站默认）

```
--bg:        主背景色
--surface:   卡片/面板底色
--surface2:  次级面板底色
--border:    默认边框色
--border2:   加重边框色

--ink:       主文字色
--ink2:      副文字色
--muted:     弱文字色
--dim:       极弱/禁用色

--accent:    主强调色（操作按钮、链接等）
--positive:  正面/成功色（是/赢/涨）
--negative:  负面/失败色（否/输/跌）
--warning:   警告/高亮色（龟币、提示）
--info:      信息色（次要强调）

--shadow-sm: 轻阴影
--shadow-md: 中阴影
--shadow-lg: 重阴影
```

### 2.2 暗色主题（对战详情页 / 开战广场）

```
--bg:        深底色
--surface:   面板 L1
--surface2:  面板 L2
--surface3:  面板 L3
--border:    半透明白边框
--border2:   加重白边框

--ink:       主文字色（浅色）
--muted:     弱文字色
--dim:       极弱色

--faction-a: 阵营 A 色（如蓝方）
--faction-b: 阵营 B 色（如绿方）
--gold:      金色强调
--danger:    危险/红色
```

> 两套主题通过 CSS 变量切换，保证所有组件只引用变量名、不硬编码色值。

## 三、主站页面结构（index.html）

### 3.1 整体布局

```
┌──────────────────────────────────────────────┐
│ 顶部导航栏 (sticky)                            │
├────────┬─────────────────────────────────────┤
│ 左侧栏  │            主内容区                   │

  ≤ 1050px → 隐藏左侧栏, 单栏布局
```

### 5.2 龟势系统（核心机制）

**体积公式：**
```
displayPower = normalize( log(totalStake + 1) × α + sqrt(totalLikes) × β )

α = 0.6 (下注权重)
β = 0.4 (点赞权重)
```

**龟缩放规则：**
- 基础大小：`scale(1.0)`
- 最大大小：`scale(2.5)` — 2.5 倍放大
- 弱势方最小比例：40%
- 使用 CSS `transform: scale()` 平滑过渡
- 两龟面对面（一方 `scaleX(-1)` 翻转）

**龟势占比计算：**
```
frPct = round(frPower / (frPower + brPower) × 100)
brPct = 100 - frPct
```

**点赞 Pips（进度圆点）：**
- 5 个小圆点
- 随点赞填充阵营色
- 满 5 个后 "maxed" 不可点击
- 满时触发 toast "赞数满了，龟势增强！"

**反驳（Rebut）系统：**
- 每条评论有 ⚔️ 反驳按钮
- 点击展开 inline textarea
- 提交后嵌套显示在原评论下方
- 反驳标记：左侧竖线 + ⚔️ badge

**阵营切换：**
- 只能在自己阵营发帖
- 对方阵营锁定，可点击切换
