### 3.2 对战系统（庄家赌局）
> 工时: 6 人天

**核心逻辑：** 1v多庄家制。发起者做庄，设定议题和立场，所有加入者自动站对立面。平台不抽成。支持公开/私人两种赌局。

**赌局类型：**
- **私人赌局：** 仅通过邀请链接或好友列表邀请加入，**不收入场费**
- **公开赌局：** 出现在赌局广场，所有用户可浏览并加入，**收取5%入场费给庄家**

**流程：**
1. 庄家创建赌局 → 设定议题 + 庄家立场 + 庄家押注 + 公开/私人 + **结算时间**（必填）
2. 私人：分享邀请链接/好友邀请；公开：自动出现在赌局广场
3. 挑战者加入 → 自动站对立面 + 下注（公开赌局扣5%入场费，私人不扣）
4. 挑战者冻结总额不超过庄家押注（满额自动封盘）
5. 进行中仍可加入（直到满额/截止/庄家手动封盘）
6. 到达结算时间 → 自动封盘，庄家**24h内**宣布结果（超时则庄家判输）
7. 结果推送给每位挑战者 → 24h内选择"通过"或"异议"（超时视为通过）
8. 全部通过 → 自动结算；任一人异议 → 提交证据，管理员仲裁（三种裁决）

**结果判定机制（庄家宣布 + 挑战者确认 + 仲裁）：**
- 庄家创建赌局时**必须设定结算时间**（议题结果揭晓的时间点）
- 到达结算时间后，赌局自动封盘，进入 pending_result 状态
- **庄家24h内宣布结果**（庄家赢/庄家输），推送给每位挑战者
- **庄家超时未宣布（24h）→ 庄家直接判输**，按庄家输结算
- 庄家宣布后，每位挑战者有**24h**操作窗口：
  - 手动选择"通过" → 该挑战者确认完毕
  - 24h内未操作 → 视为自动通过
  - 选择"异议" → 需提交理由和证据
- **全部通过**（手动+自动）→ 按庄家宣布的结果执行结算
- **任一人异议** → 赌局进入 disputing 状态，需提交事件描述+证据给人工审核

**人工审核三种裁决：**
1. **庄家赢（异议无理）** → 按庄家赢正常结算 + 提出异议的挑战者扣冻结额**10%**罚金（销毁）
2. **庄家输（庄家虚报）** → 按庄家输正常结算 + 庄家扣押注**10%**罚金（销毁）+ 庄家**禁止坐庄7天**
3. **无法判定（议题模糊）→ 赌局作废**：
   - 全额退还挑战者冻结额（挑战者不罚）
   - 庄家扣押注**10%**罚金（销毁）+ 庄家**禁止坐庄7天**
   - 庄家剩余押注退还
   - 公开赌局入场费不退

**赌局不可取消/修改规则：**
- 庄家设定赌局后**不可修改议题、立场、押注、结算时间**
- 赌局一旦有挑战者加入，**不可取消**
- 仅在 open 状态且**无任何挑战者**时可取消（退还庄家押注）

**无人挑战：**
- 到达结算时间时无任何挑战者 → 赌局自动取消，庄家押注全额退还

**入场费机制（仅公开赌局）：**
- 公开赌局：挑战者下注时，5%立即转入庄家账户（不冻结，直接到账），剩余95%冻结
- 私人赌局：不收入场费，100%冻结参与赌局
- 入场费不退（即使赌局取消）
- 这是庄家开设公开赌局的激励

**下注规则：**
- 庄家押注：100 ~ 10000 龟币（即庄家最多赔这么多）
- 挑战者单人：50 ~ 5000 龟币（公开赌局实际冻结 = 下注额×95%，私人赌局冻结100%）
- 所有挑战者冻结总额 ≤ 庄家押注（保证庄家赔得起）
- 庄家押注冻结，取消/超时全额退还

**结算公式：**
```
公开赌局: 挑战者冻结额 = 下注额 × 0.95，入场费5%到庄家
私人赌局: 挑战者冻结额 = 下注额 × 1.00，无入场费

庄家赢 → 庄家获得：本金退回 + 所有挑战者冻结额（+ 公开赌局已收入场费）
庄家输 → 每位挑战者获得：冻结额退回 + 庄家押注 × (我的冻结额 / 挑战者冻结总额)
```
- 缩头乌龟特效：输方仍获赢方分成的50%

**示例（公开赌局）：**
```
庄家押1000，3个挑战者分别下注200/400/452
入场费: A付10, B付20, C付22.6 → 庄家立即收到52.6
冻结额: A冻190, B冻380, C冻430 → 总冻结1000，满额封盘

庄家赢 → 庄家净赚1000 + 52.6(入场费)
庄家输 → A拿回190+赚190, B拿回380+赚380, C拿回430+赚430
```

**示例（私人赌局）：**
```
庄家押1000，2个好友分别下注400/600
无入场费，冻结额: A冻400, B冻600 → 总冻结1000，满额封盘

庄家赢 → 庄家净赚1000
庄家输 → A拿回400+赚400, B拿回600+赚600
```
